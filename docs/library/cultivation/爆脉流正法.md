## 摘要

本卷详述体修·爆脉流之修炼法门、真元流量公式、过载撕裂机制与经脉修补消耗。

爆脉流是末法残土最古老也最极端的战斗流派。其核心理念为：不依赖外物，将肉体经脉作为唯一的武器与防具。通过主动撕裂经脉换取瞬间的超额真元流量，在零距离将真元灌入敌人体内——一击定生死。

本卷核心设定问题：
1. 如何用量化公式表达"爆脉"的伤害与代价？
2. 经脉损伤如何影响玩家后续战斗与修炼？
3. 修补经脉需要哪些资源、如何与服务器经济系统对接？

## 正文

### 一、流派总纲

爆脉流不属于任何宗门传承。它是末法时代自然选择的结果——当灵矿枯竭、法器断档，当暗器材料越来越难获取，当阵法需要大量时间布置而你没有时间——你只剩下自己的经脉。

**核心理念**：真元流量的安全上限是枷锁。爆脉者主动打破枷锁。

**流派特征**：
- 所有技能围绕"过载"机制
- 不依赖武器耐久，不消耗载体材料
- 每次高强度战斗后必须休养经脉
- 境界越高，过载的收益与风险同步放大
- 理论上限极高——凝脉期爆满脉可一击重创固元

### 二、真元流量与安全阈值

**基础概念**：

修士的灵脉如同干枯的河床。真元在其中流动时，河床能承受的流速存在上限。

| 境界 | 真元上限 | 安全流量（点/秒） |
|------|---------|------------------|
| 醒灵 | 10 | 2 |
| 引气 | 30 | 5 |
| 凝脉 | 80 | 12 |
| 固元 | 150 | 25 |
| 通灵 | 300 | 50 |
| 化虚 | 500 | 80 |

**安全流量**：玩家每秒可不付出任何代价调动的最大真元量。普通攻击、移动、基础术法都在此范围内。

**超出安全流量的代价**：每超出1点/秒，经脉承受等量的"撕裂累积"。当累积超过阈值，触发经脉损伤。

### 三、过载撕裂机制

**爆脉公式**：

```
过载流量 = 玩家主动选择的真元输出量（必须 > 安全流量）
过载倍数 = 过载流量 / 安全流量

撕裂累积 = (过载流量 - 安全流量) × 持续时间(秒) × 脆弱系数

脆弱系数（境界越低经脉越脆弱，高阶因反复撕裂产生"疤痕硬化"）：
  醒灵: 2.0
  引气: 1.5
  凝脉: 1.2
  固元: 1.0
  通灵: 0.8
  化虚: 0.6
```

**撕裂阈值与损伤等级**：

| 累积撕裂 | 损伤等级 | 效果 |
|---------|---------|------|
| < 安全流量×2 | 无损伤 | 休息30秒自然恢复 |
| 安全流量×2 ~ ×5 | 轻度撕裂 | 真元上限临时-20%，持续游戏内2小时 |
| 安全流量×5 ~ ×10 | 中度撕裂 | 真元上限临时-40%，真元回复速度-50%，持续游戏内6小时 |
| > 安全流量×10 | 重度撕裂 | 真元上限临时-60%，无法自然回复真元，持续游戏内24小时 |
| > 安全流量×15 | 经脉断裂 | 真元上限**永久**扣除10点，境界可能跌落 |

**设计意图**：
- 轻度撕裂是"常规消耗"——爆脉者每次认真战斗后都会有的状态
- 中度撕裂意味着"接下来半天我得躲着走"
- 重度撕裂是"一周内我不想再打架"
- 经脉断裂是永久代价——只有极端情况或计算失误才会触发

**示例计算**：

> 凝脉期修士（安全流量12点/秒）
> 选择瞬间爆发：单次攻击调集30点真元，持续0.5秒
>
> 过载流量 = 30
> 过载倍数 = 30/12 = 2.5倍
> 撕裂累积 = (30-12) × 0.5 × 1.2 = 10.8
> 安全流量×2 = 24
> 10.8 < 24 → 无损伤
>
> ——这是一次"安全"的小爆发。

> 同一修士，选择全力一击：调集80点真元（全部真元池），持续0.3秒
>
> 撕裂累积 = (80-12) × 0.3 × 1.2 = 24.48
> 24.48 ≥ 24 → 轻度撕裂
>
> 打完后真元上限从80降至64，持续2小时。

> 同一修士，拼命：连续3次80点爆发，每次0.3秒
>
> 总撕裂 = 24.48 × 3 = 73.44
> 安全流量×10 = 120
> 73.44 < 120，但接近 → 中度撕裂
>
> 真元上限降至48，回复速度减半，6小时内形同半废。

### 四、爆脉技能体系

爆脉流没有传统意义上的"技能书"。其能力通过**过载在不同部位的经脉实现不同效果**。

| 技能名 | 过载部位 | 真元消耗 | 最小过载倍数 | 效果 |
|--------|---------|---------|-------------|------|
| **崩拳** | 手臂三阳经 | 20~50 | 1.5× | 零距离真元灌注。击中时，过载流量×1.2转化为对敌真元伤害。 |
| **贴山靠** | 躯干任督 | 30~80 | 2.0× | 肩撞。击退+打断敌方当前动作。自身也受击退反冲。 |
| **血崩步** | 双腿经脉 | 15~30 | 1.3× | 短距冲刺（5格），速度极快，可穿人。落地时经脉承受额外撕裂。 |
| **逆脉护体** | 全身 | 40~100 | 2.5× | 被击中瞬间激活。将敌方灌入的真元与自身过载真元对冲，抵消60%真元伤害。血肉承受剩余冲击（掉血不掉真元）。 |
| **燃命** | 丹田核心 | 全部剩余真元（≥50） | 3.0× | 以自身为媒介引爆全部剩余真元——真元不离体，而是在体内经脉中同时过载所有脉路，冲击波从肉身直接传导至3格内所有接触面。对周围目标造成（真元量×0.8）的真元伤害。使用后真元归零，必触发中度以上撕裂。（注：这不违反"真元离身即散"——燃命的真元始终以施术者肉身为导体，未暴露于空气。） |

**技能习得**：
- 崩拳：引气期自动领悟（首次真元见底时触发叙事）
- 贴山靠：凝脉期 + 累计承受100点伤害后自动领悟
- 血崩步：累计触发轻度撕裂5次后自动领悟
- 逆脉护体：累计触发中度撕裂3次后自动领悟
- 燃命：固元期 + 经历一次濒死（生命<10%）后自动领悟

### 五、经脉修补

撕裂的经脉不会自然愈合。必须使用特定资源修补。

**修补方式与消耗**：

| 损伤等级 | 自然恢复时间 | 主动修补消耗 | 修补后效果 |
|---------|------------|-------------|-----------|
| 轻度撕裂 | 游戏内2小时 | 止血草×3 + 枯荣花×1 | 立即清除损伤 |
| 中度撕裂 | 游戏内6小时 | 凝脉散×1 + 赤髓草×2 | 立即清除损伤 |
| 重度撕裂 | 游戏内24小时 | 固元丹×1 + 异变兽核×1 | 立即清除损伤 |
| 经脉断裂 | **无法自然恢复** | 通灵丹×1 + 异变兽核×3 + 在灵眼处打坐30分钟 | 恢复扣除的真元上限（境界不恢复，需重新突破） |

> **注**：通灵丹为本卷推演产物，worldview.md 中未定义。其炼制配方与获取途径需另卷补完（建议归入修行藏·丹方类）。

**修补动作**：
- 玩家需在安全位置（如灵龛）使用修补物品
- 引导时间：轻度10秒 / 中度30秒 / 重度60秒 / 经脉断裂30分钟
- 引导期间可被中断，中断后材料消耗但不生效

**经济考量**：
- 轻度撕裂的修补成本约5骨币
- 中度撕裂约25骨币
- 重度撕裂约80骨币
- 经脉断裂的修补成本超过200骨币——对于大多数玩家，不如重开

### 六、流派优势与劣势

**优势**：

1. **零启动成本**：不需要骨刺、不需要阵法材料、不需要毒草。醒来就能打。
2. **爆发上限极高**：凝脉期爆满脉的一击，真元伤害可达80×1.2=96点。固元期修士真元池150，吃满这一拳直接过半。
3. **不惧负灵域**：负灵域反吸真元的速度与境界正相关，但爆脉者的战斗方式是在进入负灵域**之前**就把真元打完。空真元的爆脉者在负灵域反而不怕被吸。
4. **心理威慑**：追杀一个爆脉者时永远要忌惮——他随时可能撕裂经脉和你同归于尽。

**劣势**：

1. **战后真空期**：每次认真战斗后必须休养。连续作战能力极差。
2. **资源消耗隐性但真实**：修补经脉的灵草消耗，长期下来不比重狙的骨刺便宜。
3. **对群弱势**：除燃命外无AOE能力。被多人围攻时只能换一个。
4. **计算失误的代价是永久性的**：经脉断裂是真永久惩罚。

### 七、与其他流派的交互

**爆脉流 vs 器修重狙**：
- 器修在50格外开枪时，爆脉者毫无办法
- 一旦爆脉者贴脸（血崩步），器修只有蜕壳或等死
- 经典博弈：器修能否在爆脉者近身之前打中那一枪

**爆脉流 vs 绝灵涡流**：
- 绝灵涡流克一切远程真元攻击
- 但爆脉者是**把真元直接塞进体内**——涡流无效
- 爆脉流是绝灵流的天敌

**爆脉流 vs 毒蛊流**：
- 毒蛊的脏真元需要时间生效
- 爆脉者追求秒杀——在毒生效前结束战斗
- 但如果爆脉者没能秒掉毒蛊流，战后撕裂+中毒=双倍痛苦

**爆脉流 vs 爆脉流**：
- 谁先出手谁赢
- 同时出手→两人一起撕裂经脉，战后各自治伤
- 末法时代最悲壮的对决：两个爆脉者互拼崩拳，同时倒地，谁先爬起来谁活

### 八、与 Bevy 系统的对接

**核心组件**：

```rust
// server/src/cultivation/body_refiner.rs

#[derive(Component)]
pub struct MeridianSystem {
    /// 安全流量（点/秒），由境界决定
    pub safe_flow_rate: f32,
    /// 脆弱系数（境界越低越大，高阶因疤痕硬化而降低）
    pub fragility: f32,

    /// 当前撕裂累积
    pub current_tear: f32,
    /// 当前损伤等级
    pub damage_level: MeridianDamage,

    /// 真元上限惩罚（比例，0.0~1.0）
    pub max_spirit_penalty: f32,
    /// 回复速度惩罚（比例）
    pub regen_penalty: f32,
}

#[derive(Component, Clone, Debug, PartialEq)]
pub enum MeridianDamage {
    None,
    Mild { remaining_secs: f32 },
    Moderate { remaining_secs: f32 },
    Severe { remaining_secs: f32 },
    Ruptured { permanent_penalty: i32 },
}

/// 爆脉技能事件
pub struct OverloadEvent {
    pub caster: Entity,
    pub part: MeridianPart,
    pub spirit_amount: i32,
    pub duration_secs: f32,
}

pub enum MeridianPart {
    Arm,   // 崩拳
    Torso, // 贴山靠
    Legs,  // 血崩步
    Whole, // 逆脉护体
    Core,  // 燃命
}
```

**过载撕裂计算系统**：

```rust
// server/src/cultivation/overload_system.rs

pub fn overload_tear_system(
    mut overload_events: EventReader<OverloadEvent>,
    mut meridian_query: Query<&mut MeridianSystem>,
    spirit_query: Query<&SpiritPool>,
) {
    for event in overload_events.iter() {
        let Ok(mut meridian) = meridian_query.get_mut(event.caster) else {
            continue;
        };
        let Ok(spirit) = spirit_query.get(event.caster) else {
            continue;
        };

        if spirit.current < event.spirit_amount {
            continue;
        }

        let overflow = event.spirit_amount as f32 - meridian.safe_flow_rate;
        if overflow <= 0.0 {
            continue; // 未过载，无撕裂
        }

        let tear = overflow * event.duration_secs * meridian.fragility;
        meridian.current_tear += tear;

        // 判定损伤等级
        let sf = meridian.safe_flow_rate;
        meridian.damage_level = if meridian.current_tear >= sf * 15.0 {
            MeridianDamage::Ruptured { permanent_penalty: 10 }
        } else if meridian.current_tear >= sf * 10.0 {
            MeridianDamage::Severe { remaining_secs: 24.0 * 3600.0 }
        } else if meridian.current_tear >= sf * 5.0 {
            MeridianDamage::Moderate { remaining_secs: 6.0 * 3600.0 }
        } else if meridian.current_tear >= sf * 2.0 {
            MeridianDamage::Mild { remaining_secs: 2.0 * 3600.0 }
        } else {
            MeridianDamage::None
        };
    }
}
```

### 九、残卷引语

**《爆脉真解·残页》**（发现于北荒坍缩渊）：
> "世人皆谓经脉不可逆。愚。经脉如弓弦，拉满方知力道。断了，换一根便是。末法之前，有体修换过七次经脉，最终以凡躯证道。"

**某化虚境爆脉者的遗刻**：
> "我这一生，崩拳打过通灵，贴山靠撞过道伥，血崩步追过天劫。最后经脉断了十三处。值吗？值。至少他们记住我了。"

**天道叙事（玩家首次触发过载时）**：
> "你感到经脉中涌过一股不该有的洪流。像干裂的河床被洪水冲刷——疼痛，但充满力量。你知道这股力量有代价。你选择支付。"

**天道叙事（玩家首次触发经脉断裂时）**：
> "一声轻响，从体内传来。不是骨骼，是更深处的东西。你的灵脉，在某一个点，彻底断了。天道没有任何表示。这种事，它见得太多了。"
