## §0 设计轴心

- [ ] 阵法 = **空间持久增益 / 陷阱 / 欺天工具**
- [ ] 物理本质：真元在方块上"刻"出临时镜印（次级投影）
- [ ] 末法约束：方块没有心镜活性 → 镜印随时间被天地噬散（"预埋数时辰朽坏"）
- [ ] 可见 / 隐蔽 / 可被破解（阵眼）
- [ ] **不做传送阵**——违反末法地理设定

## §1 第一性原理（烬灰子四论挂点）

- **影论·方块镜面**：方块本身没心镜，但作为稳定的物理结构可承载"次级投影"——前提是阵法师用真元在方块上"刻"出一个临时镜印（像水流冲出河床的轨迹）
- **缚论·阵法朽坏**：方块没有心镜的活性，所刻镜印会随时间被天地噬散（噬论的物理）。这就是"预埋数时辰朽坏"的根本原因
- **音论·欺天阵**：阵法可发出和真元相反的"音"，模拟假灵气波动（worldview §七的欺天阵核心——向天道广播假劫气权重）
- **噬论·聚灵阵悖论**：聚灵阵聚得越凶 → 局部灵气密度越高 → 越触发天道阈值 → 灵气归零 + 道伥刷新

## §5 布阵 / 触发 / 流程

```
布阵：
  选阵图 → 选点 → 消耗材料 → 布阵动画 →
  阵成（阵旗/阵眼实体）→ mirror_imprint = qi_invest × (载体系数)

触发：
  进入触发 / 主动触发 / 条件触发（如"敌真元低于 20%"）
  效果叠加规则
  与玩家 StatusEffect 的交互

朽坏 tick：
  每分钟 imprint -= (1 / 载体系数)
  imprint <= 0 → 阵法失效 + 真元蒸发

破阵：
  寻找阵眼（神识感知 / 视觉线索）
  破阵技能 / 阵图识别
  失败代价（反噬：污染 +0.5 + 经脉 MICRO_TEAR）
```

## §6 反噬 / 失败代价

- [ ] 阵成失败 = 全部材料废 + 真元 50% 蒸发
- [ ] 阵被破解 = 阵心反噬布阵者（污染 +0.5 + 经脉 MICRO_TEAR）
- [ ] **聚灵阵触发天道阈值** → 全区灵气归零 + 道伥刷新
- [ ] 无人上套（预埋数时辰）→ 真元随载体朽坏白白流失
- [ ] **欺天阵被识破** → 天道注意力反而转向布阵者（高阶博弈）

## §7 克制关系

- **克**：所有移动型修士（在通道布陷阱）；不熟地形者
- **被克**：神识感知 / 阵图识别破解；毒蛊（阵法师布阵全神贯注，易被针刺）；爆脉流（直接打碎载体）
- **染色亲和**：缜密色（阵法流原生匹配，加成幅度待定）；凝实色载体保镜印更稳（幅度待定）
- **错配**：沉重色（体修）走阵法——心思粗，阵图易出错
