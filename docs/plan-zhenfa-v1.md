## §1 第一性原理（烬灰子四论挂点）

- **影论·方块镜面**：方块本身没心镜，但作为稳定的物理结构可承载"次级投影"——前提是阵法师用真元在方块上"刻"出一个临时镜印（像水流冲出河床的轨迹）
- **缚论·阵法朽坏**：方块没有心镜的活性，所刻镜印会随时间被天地噬散（噬论的物理）。这就是"预埋数时辰朽坏"的根本原因
- **音论·欺天阵**：阵法可发出和真元相反的"音"，模拟假灵气波动（worldview §七的欺天阵核心——向天道广播假劫气权重）
- **噬论·聚灵阵悖论**：聚灵阵聚得越凶 → 局部灵气密度越高 → 越触发天道阈值 → 灵气归零 + 道伥刷新

### 3.1.A 载体（Q13: C 受限白名单 + 开放降级）

**白名单外**：任意方块都可封（开放降级），统一按"普通石块"规格处理（30min / 10%）。**地师懂地，但平地起雷也是手艺**。

### 3.1.B 触发（Q17: B 踩雷 + 主动触发）

- **踩雷**（被动）：实体踩入触发格 → 立刻引爆
- **主动触发**（主动）：玩家手持阵旗 / 阵图按特定键引爆自己埋的任意雷
  - **距离限制**：神识感知范围内（按境界递增——具体范围接 plan-perception-v1）
  - **不消耗额外真元**：已封进载体的真元就是触发能量
- **条件触发**（"敌真元 <20% 自动炸"等 AI 化）：留 vN+1

### 3.1.C 真元投入（Q18: C 玩家自选 + 载体上限）

埋雷时玩家在 `[5% qi_max, 载体承载上限]` 区间自选投入比例（UI: 滑条或预设档 5 / 10 / 20 / 30 / 50%，受载体 cap 限制）。

**基础伤害公式**：

```
damage_tier = wound_mapping(qi_invest_ratio)
范围 = range_mapping(qi_invest_ratio)
```

按 worldview §四 三层战力档次映射（不直接给数字，给定性档）。**Q20: B 已拍**——投入越大范围越广：

| 投入比例（% 当前 qi_max）| 范围 | 腿部伤口档 | 腿部经脉损伤 |
|---|---|---|---|
| < 5% | 单格 | BRUISE | INTACT |
| 5% - 15% | 单格 | LACERATION | MICRO_TEAR |
| 15% - 30% | **3×3** | FRACTURE | TORN |
| > 30%（仅夜枯藤 / 兽核可达） | **5×5** | SEVERED | SEVERED |

> 30% 投入是地师的"小型全力一击"——孤注一颗夜枯藤雷可断小队 5 人之腿。但代价：**该雷一次性消耗 30% 当前真元池**，命中前自身脱力（worldview §四 全力一击虚脱原则）。范围扩大 + 投入加大双管齐下，让 zhenfa 的"赌注感"在两个维度同步放大。

### 3.1.D 伤害形态（Q19: B 基础 + 染色/特性定向）

**基础形态（无特性）**：
- 地刺贯穿目标所在格的腿部（worldview §五.3 锚定）
- 腿部伤口档 + 腿部经脉损伤（按 3.1.C 公式）

**特性变种**（玩家自身具备对应特性时自动叠加，不分流派——遵循 worldview §五"特性偏泛型"正典）：

| 特性 | 诡雷变种效果 |
|---|---|
| **毒性真元** | 触发后腿部经脉污染 +X（X 按 qi_invest 比例） |
| **暴烈色** | 触发时额外击退数格 + 短暂麻痹（1s） |
| **凝实色** | 镜印朽坏速度 -50%（增益型，非伤害变种）|

无特性玩家 → 纯基础形态。多个特性叠加按各自独立加权（不互斥）。

#### 地师对立天赋——隐藏 vs 识别

**核心机制**：地师可通过顿悟（worldview §六.3 Category::Style）选择 **"藏阵"** 或 **"破阵"** 两条对立成长路径。两条天赋互斥——专精一边，另一边自然退化。

| 天赋 | 效果方向 | 升级方式 |
|---|---|---|
| **藏阵**（Concealment）| 自己埋的诡雷被识色法看到的概率/范围下降；缜密色识阵者看自己的雷也变难 | 顿悟选"藏"路径，每次累积 +X 等级 |
| **破阵**（Disenchant）| 识色法识别他人诡雷的范围 +X、窗口时间 +Y%、拆除归还概率 +Z%（参 §3.1.E 拆除收益）| 顿悟选"破"路径，每次累积 +X 等级 |

**对立性原则**（呼应 worldview §六 "杂色失效"）：

- 累积选 5 次顿悟形成"专精"——选 5 次"藏"则破阵能力反而衰退（数值待定）
- 二者不可同时高等级，强迫地师选"我是埋雷的还是破雷的"
- 顿悟事件是稀缺资源（一生 5-10 次），玩家需慎重选择投入哪边

**实装锚点**：
- server `cultivation::insight::InsightCategory::Style` 扩展 `StyleEffect::ZhenfaConcealment / ZhenfaDisenchant` variant（前置：之前 subagent 调研的 Gap 1 "StyleEffect 白名单未实装"——本 plan P0 顺手补一份）
- 玩家累积等级写入 `Cultivation` 或新 component `ZhenfaSpecialization`

#### 破阵流程

发现阵眼后玩家可选两条路径：

| 路径 | 操作 | 代价 | 收益 |
|---|---|---|---|
| **A. 拆除** | 在阵眼前持续 inspect 3-5s | 消耗少量真元 | **概率归还破阵者**（封存真元）—— 见 Q25 拍板 |
| **B. 强行破坏载体** | 直接打碎载体方块 | **触发反噬**（污染 +0.5 + 经脉 MICRO_TEAR）| 即刻摧毁，但要扛一记反噬伤害 |

**反噬来源解释**：阵眼承载的镜印强行解构 → 残余真元逆涌入破阵者经脉（worldview §四 异体排斥 + §三 镜印朽坏的物理）。

#### 拆除收益（Q25: B 概率归还 + 经验叠加）

拆除成功后，封存真元按概率回流破阵者背包（以"散逸真元珠"或类似 item 形态发放，可塞回自身真元池或转赠）：

| 维度 | 数值 | 说明 |
|---|---|---|
| 基础归还概率 | **30%** | 破阵者首次上手，多数情况"巧手不足，真元散尽" |
| 经验加成 | 每级 +5-10% | 接 plan-skill-v1 新增 `SkillId::Disenchant` / `SkillId::Zhenfa` 派生（命名待定）|
| 概率上限 | **80%** | 永远保留 20% 风险——symbol of 末法易挥发（worldview §三）|
| 失败时 | 蒸发归无 | 真元被天地噬散，谁也得不到 |

**设计意图**：把"破阵"做成长期成长指标——新手玩家拆雷常空手而归（30% 中），但反复破阵积累经验后变身"破阵高手"（80% 拿），收益曲线契合 zhenfa 的"算计"主题。同时保留 20% 终身风险，让破阵不会变成无本买卖。

### 3.1.F 多雷叠加（Q22: C 同一格仅 1 颗 + 邻接连锁触发）

#### 叠加规则

- **同一格**：仅允许 1 颗诡雷。第二颗在已埋格不能埋（提示"此地已有阵眼"）
- **邻接 3×3 内**：可埋多颗，触发时按"连锁规则"依次引爆

#### 连锁规则

- 任一诡雷被触发时（踩雷 / 主动触发），其邻接 3×3 范围内的**所有未触发**诡雷在 **0.3s 延迟**后**依次连环引爆**
- 连锁顺序：按距离排序（近 → 远），同距离按布阵时间顺序
- 已触发的诡雷标记 `triggered: true`，不会被二次触发，避免循环
- **跨阵眼连锁**：A → B 触发时，若 B 的邻接 3×3 内还有 C，C 也会被引爆——理论上可形成"链式爆破带"，是 zhenfa 玩家"布连环阵"的核心玩法

#### 连环阵示例

```
玩家在 5×5 区域埋了 4 颗雷：

. . . . .         踩中 X 后 0.3s：A 爆 (X 是 A 邻接 3×3 内)
. A B . .         A 爆后 0.3s：  B 爆 (B 是 A 邻接 3×3 内)
. . X . .  →      B 爆后 0.3s：  C 爆 (C 是 B 邻接 3×3 内)  
. . C D .         C 爆后 0.3s：  D 爆
. . . . .         合计 1.5s 内 5×5 范围全炸
```

#### 实施约束

- server `ZhenfaRegistry.find_neighbors(pos, radius=1)` 查询邻接
- 每颗 `ZhenfaInstance` 持有 `triggered_at: Option<Tick>`
- 客户端粒子按服务端 emit 的时序播放，避免一次性卡顿

### 3.1.G 主动触发 UI（Q23: D 按键引爆最近 + Shift 列表精选）

**前置条件**：玩家**手持阵旗 / 阵图 item**才能主动触发——空手不可触发，强迫地师"出门带工具"。阵旗作为常规 item 走 plan-inventory-v1 物品栏。

#### 触发模式

| 模式 | 操作 | 行为 |
|---|---|---|
| **快引爆** | 持阵旗 + 按键（默认 K，可在 keybindings 改）| 引爆神识感知范围内**最近**一颗未触发的诡雷；无候选时提示"识海无阵息" |
| **精选引爆** | 持阵旗 + Shift+按键 | 打开 `ZhenfaIgnitionScreen`：列表显示自己埋的所有诡雷（坐标 + 载体类型 + 真元投入比例 + 朽坏剩余时间 + 距离当前位置）→ 玩家点选某颗引爆 |

#### 实施约束

- server `ZhenfaRegistry` 提供 `find_owned_by(player_id) -> Vec<ZhenfaInstance>`
- 神识感知范围 = `spiritual_sense` 范围（plan-perception-v1.1 已实装的 `SenseRadius`）
- `ZhenfaIgnitionScreen` 走 `DynamicXmlScreen` 已实装基础设施（参 plan-zhenfa §8 已记录）
- "最近"按欧氏距离排序；同距离取最早埋设的（FIFO）

#### 边界情况

- 阵旗丢失 / 损坏 → 玩家失去主动触发能力，但已埋的雷仍按踩雷被动正常工作
- 玩家死亡 → 阵旗按 worldview §十二死亡掉物规则掉地；新持有者不会自动获得"引爆他人雷"的权限（owner-only 校验，参 §8 数据契约）
- 阵旗失效 / 多阵旗 → 一名玩家同时只能持一面有效阵旗（多面取手中那面）

## §3.2 警戒场·v1 规格（P1 阶段）

> **区域嵌入式识色法**——把 §3.1.E 的"识阵"能力延伸到玩家不在场的远程监视。跟诡雷构成"埋伏 + 监视"组合战术：诡雷负责伤害，警戒场负责报信。

**Primary Axis 复用**：跟诡雷一样依赖"真元逆逸散效率" —— 但语义不同。诡雷是"封存能量等爆"，警戒场是"封存识海等响"。

### 3.2.A 载体（复用诡雷白名单）

跟 §3.1.A 同一套白名单方块 + 开放降级，**单点中心阵眼**模型：1 格白名单方块即"阵眼"，监听半径由投入决定。承载上限同诡雷：

| 载体 | 镜印保持 | 真元承载上限 | 来源 |
|---|---|---|---|
| 普通石块（开放降级基线） | 30 分钟 | 10% qi_max | 自采 |
| 灵气浸润方块 | 2 小时 | 20% qi_max | worldgen 馈赠区 |
| 夜枯藤 | 12 小时 | 30% qi_max | ecology-0002 |
| 异变兽核镶嵌方块 | 24 小时 | 50% qi_max | peoples-0005 |

**与诡雷的载体互斥**：同一格方块要么是诡雷阵眼要么是警戒场阵眼，不可同时双用——避免 stack 复杂度。

### 3.2.B 触发（Q26: A 任何实体都触发）

**进入触发**：任何实体（玩家 / NPC / 异变兽 / 动物 / 飞行物）进入监听半径触发一次"入侵通知"——不区分敌我友。

**为什么不过滤**：
- 末法残土风吹草动皆可疑，玩家自己判断"这次响是值得理会还是路过的兔子"
- 不依赖 plan-social Faction / Hostility 判定 API（解耦）
- 假阳性是 zhenfa 玩家"识场感"的一部分——经验丰富的地师能从警戒频率判断地形是否平静

**节流机制**：
- 同一实体反复进出区域 → 60s 内仅触发 1 次
- 多实体同时进入 → 每实体单独计入，仅按"进入"那一刻为准

**离开 / 持续判定**：v1 不做"X 留在区域内"的持续报警——只报"进入"事件。vN+1 可加。

### 3.2.C 真元投入与监听半径（Q27: B 投入决定大小）

跟诡雷 §3.1.C 同一套"自选投入"机制，监听半径随投入扩大：

| 投入比例（% 当前 qi_max）| 监听半径（球面 R）| 监听格数 ~ |
|---|---|---|
| < 5% | R=3 | ~33 格 |
| 5%-15% | R=5 | ~92 格 |
| 15%-30% | R=8 | ~225 格 |
| > 30%（仅夜枯藤 / 兽核可达）| R=12 | ~509 格 |

> 30% 投入是地师的"远视警戒"——R=12 球面对应 ~24 格直径，可监视一个小山谷的全部入口。但代价：**该警戒场一次性消耗 30% 当前真元池**，命中前自身脱力（worldview §四 全力一击虚脱原则）。

#### 地师专精系数（Q30: C 任何玩家可布 + 地师专精）

警戒场是**真元封存类阵法的通用机制**——不锁流派 unlock，**任何玩家都能布**。但地师天赋显著强化效果，遵循 worldview §五"特性偏泛型 + 流派助力"原则。

**地师专精等级**通过累计的 zhenfa 相关顿悟次数判定（即 §3.1.E 地师对立天赋系统中的 "藏阵" + "破阵" 等级和）：

| 地师等级 | 触发条件 | 监听半径系数 | 载体朽坏速度系数 |
|---|---|---|---|
| **None**（普通玩家）| 0 次 zhenfa 顿悟 | **×0.5** | ×1.5（朽坏更快）|
| **Novice**（地师入门）| 1-2 次 zhenfa 顿悟 | ×0.75 | ×1.25 |
| **Expert**（地师精修）| 3+ 次 zhenfa 顿悟 | **×1.0** | ×1.0 |

**实装**：server `is_zhenfa_specialist(cultivation) -> ZhenfaSpecLevel` 查询函数 → 系数应用到 §3.2.C 半径表 / §3.2.A 朽坏 tick。

**设计意图**：worldview §五已正典化"流派偏泛型"——任何修士都能学任何流派，匹配特性事半功倍而已。普通玩家也能用警戒场，只是覆盖小、撑不久。**跟暗器流的载体封存可被任何流派复用是同一原则**。地师专精的优势体现在"覆盖更广 + 撑更久"——量变而非质变。

### 3.2.D 通知形态（Q28: D 分层组合）

**收到通知的玩家** = 布阵者本人（owner-only），无论玩家是否在场（包括离线状态）。

#### 三层通知（各司其职）

| 层 | 形态 | 触发条件 | 设计意图 |
|---|---|---|---|
| **声音层** | client 本地短警示音（"叮——"或染色专属音色 in plan-audio-v1）| 玩家在场（同维度，触发点在 chunk load 范围内）| 立即吸引注意，不抢 HUD 空间 |
| **指示器层** | 复用 perception-v1.1 — 扩展 `SenseKindV1::ZhenfaWardAlert` 走屏幕边缘箭头 + 距离方位 | 玩家在场或邻近（神识感知范围内）| 提供入侵位置和方向 |
| **narration 层** | agent tiandao 推送一段叙事（例："你心头一颤——东南林深处，布的那片警戒场告诉你，有人来了"）| 任何状态（包括离线）| 保留在玩家 message log，事后可复盘 |

#### 离线 / 远距离场景

- 玩家**离线**或**不在场**（远距离 / 不同维度 / chunk 未加载）：
  - 声音层 ❌ 不播
  - 指示器层 → Redis pending 队列缓存（玩家上线 / 进入感知范围时回放）
  - narration 层 → 立即生成存进 player message log
- 玩家**上线**后：
  - 自动回放最近 24h 内未读 narration（按时序）
  - 指示器层仅回放最近 5-10 条（避免一次性 spam）—— 数值待 telemetry 调参（参 Q29）

#### 通知载荷（payload structure 草案）

```json
{
  "type": "zhenfa_ward_alert",
  "ward_id": "<UUID>",
  "intruder_kind": "Player|Npc|Animal|Hostile|...",
  "intruder_id": "<entity_id 或 null>",
  "position": { "x": 100, "y": 64, "z": 200 },
  "distance": 23.5,
  "direction_hint": "ESE",
  "triggered_at": "<unix_ts>"
}
```

详细 schema 见 §8 数据契约（待补）。

### 3.2.E 警戒场识色法 + 破阵（复用诡雷 §3.1.E）

跟诡雷一样**完全隐蔽**——默认不可见，识色法主动 inspect 才能看到。

| 途径 | 效果 |
|---|---|
| 基础识色法（主动） | 同诡雷 §3.1.E |
| 缜密色被动 | 同诡雷 §3.1.E |
| 地师**识别天赋**（破阵） | 同诡雷 §3.1.E（藏 vs 破对立体系）|

**特殊**：识色法 inspect 中可分辨阵眼是诡雷 vs 警戒场——两者在客户端粒子 / 染色 / 边缘指示器形态不同。

**破阵流程**同诡雷：拆除（概率归还，参 §3.1.E）/ 强行破坏载体（触发反噬：污染 +0.3 + 经脉 MICRO_TEAR；反噬比诡雷轻 40%——警戒场封存的真元密度低）。

### 3.2.F 跟诡雷的组合战术（zhenfa 灵魂循环）

警戒场 + 诡雷构成 zhenfa 玩家的核心战术循环：

```
1. 选址（狭通道 / 必经之路 / 灵草丛）
2. 埋诡雷（多颗，构成 §3.1.F 连环阵）
3. 在诡雷阵周围布警戒场（监听半径覆盖诡雷阵）
4. 玩家撤离到神识感知范围内安全位置
5. 等待警戒触发 → 收到入侵通知
6. 主动触发引爆诡雷（§3.1.G 阵旗）→ 远程精准打击
7. 撤离 / 收集战利品
```

**为什么这是 zhenfa 灵魂玩法**：
- 战斗不在场也能赢——这是地师对"末法距离衰减"（worldview §四）的破解之道：本人不在场，飞过去的不是真元而是信息
- 信息战层级——识破他人警戒场 = 知道哪片区域被布防
- 心理战层级——故意布警戒场让对方知道有人监视，逼对方改路

## §5 布阵 / 触发 / 流程

```
布阵：
  选阵图 → 选点 → 消耗材料 → 布阵动画 →
  阵成（阵旗/阵眼实体）→ mirror_imprint = qi_invest × (载体系数)

触发：
  进入触发 / 主动触发 / 条件触发（如"敌真元低于 20%"）
  效果叠加规则
  与玩家 StatusEffect 的交互

朽坏 tick：
  每分钟 imprint -= (1 / 载体系数)
  imprint <= 0 → 阵法失效 + 真元蒸发

破阵：
  寻找阵眼（神识感知 / 视觉线索）
  破阵技能 / 阵图识别
  失败代价（反噬：污染 +0.5 + 经脉 MICRO_TEAR）
```

## §6 反噬 / 失败代价

- [ ] 阵成失败 = 全部材料废 + 真元 50% 蒸发
- [ ] 阵被破解 = 阵心反噬布阵者（污染 +0.5 + 经脉 MICRO_TEAR）
- [ ] **聚灵阵触发天道阈值** → 全区灵气归零 + 道伥刷新
- [ ] 无人上套（预埋数时辰）→ 真元随载体朽坏白白流失
- [ ] **欺天阵被识破** → 天道注意力反而转向布阵者（高阶博弈）

## §7 克制关系

- **克**：所有移动型修士（在通道布陷阱）；不熟地形者
- **被克**：神识感知 / 阵图识别破解；毒蛊（阵法师布阵全神贯注，易被针刺）；爆脉流（直接打碎载体）
- **染色亲和**：缜密色（阵法流原生匹配，加成幅度待定）；凝实色载体保镜印更稳（幅度待定）
- **错配**：沉重色（体修）走阵法——心思粗，阵图易出错
