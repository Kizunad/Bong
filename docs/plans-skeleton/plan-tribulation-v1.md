# Bong · plan-tribulation-v1 · 模板

**天劫专项**。HUD `TribulationLocked` / combat §10 都留了钩子，本 plan 定义：触发条件、阶段流程、奖励/代价、失败处理。

**世界观锚点**：`worldview.md §三` 明确末法世界**唯一**的境界天劫是 **渡虚劫**（通灵 → 化虚），其他境界突破靠静坐+事件触发，**不降天劫**。本 plan 不得引入"小雷劫/金丹劫/元婴劫"等上古/传统称呼。

**交叉引用**：`plan-combat-v1.md §10` · `plan-cultivation-v1.md` · `plan-HUD-v1.md §3.3` · `worldview.md §三/§七`。

## §0 设计轴心

- [ ] 唯一境界天劫 = 渡虚劫；活下来 → 化虚，死了 → 回通灵初期
- [ ] 全服广播："有人在渡虚劫。"——所有人知道你的位置（含截胡者）
- [ ] 高风险高回报，无辅助、无捷径（worldview line 130）
- [ ] 区别：**境界天劫**（渡虚劫）vs **天道激烈手段**（域崩、对高消耗修士的针对性雷劫）

## §1 触发条件

| 类型 | 触发 | 备注 |
|---|---|---|
| 渡虚劫（通灵→化虚）| 奇经八脉全通 + 玩家主动起劫 | 唯一的境界突破天劫 |
| 域崩 | 区域灵气将尽 | 区域级，所有人必须撤离 |
| 定向天罚 | 化虚修士 / 高消耗者 | 天道精准收割（worldview §七） |

## §2 阶段流程（渡虚劫）

```
  天象预兆（远方雷云 + 全服广播） → 锁定（TribulationLocked）
  → N 波雷劫（强度递增） → 结算（化虚 / 退回通灵初期）
```

- [ ] 预兆时长（含给截胡者赶路时间）
- [ ] 锁定范围（不可逃离、不可下线）
- [ ] 波次数量与难度曲线
- [ ] 玩家可用资源：自身真元 / 已布阵法 / 提前藏好的丹药

## §3 奖励 / 代价

| 结果 | 效果 |
|---|---|
| 圆满度过 | 进入化虚境（服务器仅容 1-2 人，worldview line 72） |
| 死亡 | 退回通灵初期 + 走重生流程（见 plan-death-lifecycle-v1） |
| 被截胡杀死 | 同上，但杀手获得目击/战绩记录 |
