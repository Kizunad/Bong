**本 plan 的角色**：
本 plan 不是某个单独系统的实现 plan。它是**全程旅途的路书**——定义各阶段应该有什么、解锁什么、玩家的决策临界点在哪里。每个具体 TODO 由对应的子系统 plan 负责落地，本 plan 负责：
1. 阶段切分和节奏控制（什么时间给玩家什么）
2. 玩法解锁时序（种田什么时候能开始？阵法什么时候解锁？）
3. 新手引导钩子（第一个 30 分钟怎么留住人）
4. 中期承诺点（3h / 10h / 25h 各有什么"硬目标"让玩家觉得值）
5. 终局体验（化虚后的世界是什么样）
6. 多周目差异（重生/新角色时的体验差异）

## §0 设计轴心

- [x] 旅途总时长基线 ~100h（纯静坐 ~51.5h，其余 48.5h 为探索/战斗/采集/加工/社交/跑图/等资源）
- [x] 六阶段（0→醒灵→引气→凝脉→固元→通灵→化虚），各阶段解锁新系统/新区域/新威胁
- [x] 五条玩法线（炼器 / 炼丹 / 种田 / 阵法 / 搜打撤）在凝脉期（~3.5h）后全面分叉
- [x] 流派选择（四攻击 / 三防御）在引气→凝脉过渡期（~2-3h）开始呈现
- [x] Minecraft 开放性——玩家可以不按本 plan 推荐的顺序走，但各阶段仍有硬门槛（经脉数、真元上限、突破事件条件）
- [x] 新手前 30 分钟必须有至少一次突破（醒灵→引气）、一次危险遭遇、一处可返回的安全点
- [x] 死亡不毁旅途——每次死给遗念信息（worldview §十二），运数前 3 次安全网
- [x] 天道始终在背景说话——narration 不是教程提示，是冷漠的、古意的环境旁白

## §2 P1 · 初醒（0h → 0.5h）— "留住人"

**目标**：30 分钟内让玩家完成「感知灵气 → 打通第一条经脉 → 建立灵龛 → 突破到引气」，建立"这个世界有东西"的认知。

### 2.1 出生点体验

- [ ] **出生即醒灵**：玩家从「残土」中苏醒（一个半埋在地下的石棺状方块），周围是初醒原的低灵气荒野
  - 第一眼看到：灰白的残灰地面、枯木、远处几只噬元鼠在游荡
  - 天道第一句话："你又醒了。上一次你没撑过引气。"（如果是新角色："你醒了。天地在你眼里是灰的——那是因为你还不配看见颜色。"）
  - HUD 右下角出现灵气条（醒灵可见灵气浓淡）
- [ ] **第一步·感知灵气**：
  - 醒灵期玩家 HUD 上可以看到「灵气浓淡」——一个简易的色块指示器（白=灵气近零，淡绿=0.1-0.3，绿色=0.3+）
  - 玩家移动时灵气条变化——向绿色方向走=向馈赠区走
  - 设计钩子：玩家出生点灵气 0.1，向北走 200 格灵气升到 0.3——让玩家自然学会"跟着灵气走"
- [ ] **第一次打坐**：随地右键长按进入"静坐"状态
  - 静坐时画面色温微微变化，真元条缓慢上涨
  - 天道："你试着把天地间残存的那点东西吸进肺里。很淡。但总比没有强。"
  - 真元满 10/10 时 → 天道提示（非 UI，narration）："够了。找条经脉冲过去。"
- [ ] **第一条正经打通**：玩家在真元满时选择一条正经冲击（醒灵→引气的唯一硬门槛是 3 条正经通+灵气>0.5 处静坐 3 分钟）
  - 第一条正经打通——视觉：经脉 HUD 上第一条经脉亮起，体感字幕飘过"手太阴肺经·通"
  - 醒灵→引气突破条件达成：3 条正经通 + 灵气>0.5 处静坐 3 分钟不被打断
  - 但此时玩家还在初醒原（灵气 0.3），需要引导去找 >0.5 的灵气点

### 2.2 第一个决策点：灵龛选址

- [ ] **龛石获取**：初醒原出生点附近必定有一块「龛石」（一次性道具，worldview §十一）
  - 获得时 narration："此石可为你造一个家。但家只能有一个——选错了地方，就等死后再选。"
- [ ] **灵龛建立**：
  - 玩家在任意位置右键龛石 → 灵龛落地（方圆 5 格安全区、NPC 不主动攻击、其他玩家无法破坏方块）
  - 灵龛不提供灵气——不能在家修炼（这是核心设计：逼玩家出门）
  - 安全区视觉：地面微微泛光、色调温暖
  - narration："此地记住了你。你死的时候，会从这里重新爬起来。"

### 2.3 第一个危险遭遇

- [ ] **噬元鼠群初遇**：在去往 >0.5 灵气的路上，必定遭遇 2-3 只噬元鼠
  - 噬元鼠攻击不掉血，**直接扣真元**——玩家会惊恐发现真元条在往下掉
  - 如果真元被偷光 → 打不死鼠 → 只能逃跑
  - narration："鼠。不是来吃你——是来吃你体内那点好不容易攒起来的东西。"
  - 这是"战斗 = 真元汇率"的第一次教学（不是扣血是扣真元）
- [ ] **第一次死亡教学**（可选，取决于玩家是否被鼠群耗死）：
  - 醒灵期死 → 运数-1（还剩 2 次运数）→ 重生在灵龛（如果建了）→ 真元归零
  - 遗念："初醒原向西 500 格，有一个比你更蠢的人留下的半袋干粮。"
  - 遗念信息真实——指向一处真正有物资的地点

### 2.4 第一次突破：醒灵 → 引气

- [ ] **寻找 >0.5 灵气点**：初醒原的灵气不达标（0.3），需要往灵气更浓的方向走
  - 方向提示（通过 narration）："风从东北来，那里的空气……稍微厚一点。"
  - 东北方向通往青云残峰外围（灵气 0.5-0.7）
  - 或：初醒原偶尔会有天道投放的"微型灵气潮"（30 分钟期限的 0.6 点）
- [ ] **3 分钟静坐突破**：
  - 在灵气 >0.5 处静坐，HUD 出现突破进度条（3 分钟倒计时）
  - 这 3 分钟是**脆弱窗口**——不能动、不能战斗、被攻击打断
  - 周围可能有噬元鼠或其他玩家
  - 突破成功：视野瞬间变色——世界从灰暗变得有色彩（"你能看见灵气了"）
  - HUD 解锁：灵气精确数值（引气可见精确值）、生物气息感知（50 格内）
  - narration："你活下来了。天地又多了一个吸它血的嘴。"

## §3 P2 · 探索（0.5h → 3.5h）— "第一个承诺点"

**目标**：3 小时内让玩家体验「搜打撤完整循环」、遇到 NPC/其他玩家、打通第 2-3 条正经、第一次炼丹尝试、建立对世界规则的直觉理解。

### 3.1 搜打撤初体验

- [ ] **搜（探索）**：从初醒原向外辐射，四条方向各指向一个区域
  - 西北 → 青云残峰（灵草、矿石，~3600 格/7 分钟路程）
  - 东北 → 灵泉湿地（大量灵草、修炼圣地，~3500 格）
  - 东南 → 血谷（矿脉、灵眼不固定，~3900 格）
  - 西南 → 幽暗地穴（地下洞穴网、残卷，~3600 格）
  - 每条路线上散布荒野内容：零星灵草、噬元鼠群、灰烬蛛伏击点、偶发伪灵脉
- [ ] **荒野赶路体验**：
  - 赶路时真元自然衰减（无修炼状态下缓慢流失）
  - 路上遇到其他玩家——无名字、无标识、互相评估
  - 灰烬蛛伏击——地面上残余灰方块踩上去减速 + 留下可追踪脚印
  - narration 偶尔插话（指路/警告/嘲讽）——"东边 300 格外有个人，境界和你差不多。他的真元还剩一半。"
- [ ] **打（战斗/资源竞争）**：
  - 见到灵草 → 采集前观察周围（有没有蛛？有没有埋伏？）
  - 见到其他玩家 → 评估：打还是绕？
  - 第一次战斗（vs NPC 散修或 vs 野兽）——体验距离衰减 + 真元消耗
- [ ] **撤（撤退）**：
  - 采集完毕 → 回灵龛
  - 如果被追杀 → 利用灰烬蛛或地形甩掉追兵
  - 如果真元见底 → 必须找安全点打坐恢复
  - 关键教学：**知道什么时候跑比知道怎么打更重要**

### 3.2 第二次遭遇：拟态灰烬蛛

- [ ] **灰烬蛛伏击**：经过残灰地面时触发
  - 蛛从地面暴起攻击
  - 战斗教学三：拟态生物（外观与方块一致）——观察环境的重要性
  - 掉落：蛛丝（蜕壳流材料）、骨片（器修暗器材料）
- [ ] **老玩家的利用**：narration 提示——"你可以留着它的命。追你的人会替你踩上。"
  - 这是"环境利用 > 硬打"的第一次教学

### 3.3 NPC 初遇：拾荒散修

- [ ] **第一次遇到 NPC**：在青云残峰边缘遇到拾荒散修
  - 评估阶段：NPC 评估玩家境界和真元状态 → 决定行为
  - 如果玩家真元满 → NPC 恭敬交易，"小兄弟，拿这株星辉草换你两枚骨币"
  - 如果玩家真元见底 → NPC 翻脸，"对不住了——你身上的东西先借我用用"
  - 教学四：**状态暴露 = 弱点暴露**——真元不能随便见底
- [ ] **第一次交易**：以物易物或骨币交易
  - 双方保持 10 格安全距离
  - 物品丢在地上互换
  - narration："他给你的东西值不值，你自己看。交易完了最好换个方向走——他知道你往哪去了。"

### 3.4 第一次炼丹

- [ ] **基础配方获取**：拾荒散修或初醒原宝箱可获基础丹方（开脉丹、回元丹）
- [ ] **材料收集**：2 株灵草 + 1 份辅料
- [ ] **第一次炼丹**：在灵龛附近找个安全角落，用简易丹炉（凡铁锅 + 篝火）
  - 炼制开脉丹：加速经脉打通进度 30%
  - 可能失败（初期失败率 30-50%）——"丹炉冒出一股黑烟，灵草化成灰"
- [ ] **丹药即用即走**：不囤积——会贬值（shelflife 半衰期）

### 3.5 引气 → 凝脉 突破准备

- [ ] **经脉打通节奏**：
  - 第 2 条正经：~15 分钟静坐（可在灵龛附近的 0.3 灵气点完成）
  - 第 3 条正经：~15 分钟静坐
  - 3 正经全通后 → 引气→凝脉准备就绪
- [ ] **突破条件**：6 条正经通 + 真元池满 + 一次性灌满冲击形成局部循环（worldview §三）
  - 此时还差 3 条正经 → 留到下一阶段
- [ ] **凝脉散**：可用炼丹制作的辅助丹药，降低形成局部循环的失败率
  - 引导玩家进入下一阶段："还需要三条正经——你需要去灵泉湿地。那里有你在初醒原找不到的药材。"

## §4 P3 · 分叉（3.5h → 11.5h）— "选路"

**目标**：凝脉期。玩家开始接触全部五大玩法线，做第一次重要的流派选择，打通第 4-6 条正经，第一次有意识地"构建自己的打法"。

### 4.1 引气 → 凝脉 突破

- [ ] **寻找安全突破点**：需要灵气 >0.5 + 3 分钟不被打断
  - 可选地点：青云残峰外门、灵泉湿地边缘、或自己找到的隐蔽洞穴
  - 风险：这些地点其他玩家也知道
- [ ] **凝脉突破**：
  - 6 条正经通 + 真元满 + 一次性灌满 → 形成局部循环
  - 成功：真元上限 30 → 80、可感知区域灵气精确值
  - 失败：真元上限临时 -10（需要养伤恢复）
  - narration："你体内有了第一条循环。从今天起，你可以说自己是个修士了——虽然只是最不入流的那种。"

### 4.2 五大玩法线解锁

凝脉期 = 全面解锁。以下五条线在此阶段**同时可用**，玩家自由选择深度投入：

#### A. 炼器线（器修 / 暗器流）

- [ ] **第一次炼器**：凡铁 + 异兽骨 + 灵矿 → 第一把附灵武器
  - 需求：基础锻造台（凡铁可做）、异兽骨骼（打死缝合兽或从 NPC 交易获取）、灵矿（挖矿）
  - 最低入门品：铁剑（凡器，不附灵）→ 骨刺（封入真元，暗器流核心）→ 附灵武器（灵纹第一次激活）
- [ ] **暗器流入门**：采集异兽骨骼 + 灵木 → 制作封灵骨刺
  - 真元注入骨刺：花费 30 分钟把 80% 真元封入 3 根骨刺
  - 射程 30 格保留大部分真元（世界观 §四 距离衰减）
  - 风险：射空 = 巨亏——材料和真元都白费
- [ ] **器修中期目标**：附灵武器灵纹激活、三根骨刺常备背包、「封灵匣」制作（保管上古遗物不掉耐久，§十六.四）

#### B. 炼丹线

- [ ] **系统炼丹**：从简易篝火锅 → 正式丹炉（需要灵矿+阵法基础）
  - 丹方收集：开脉丹、回元丹、凝脉散、固元丹（需探索/交易获取）
  - 炼丹失败率：基础配方 30%、进阶配方 50%+（可用灵草质量/火候/夏冬季节影响概率）
  - 炼丹特色产出：副作用丹药（如 §九 炼丹疯子 NPC 的禁药——半小时爆发翻倍但永久-1 真元上限）
- [ ] **炼丹师真元染色**：长期炼丹 → 温润色（偏自疗、中和异种真元）
- [ ] **药材保鲜**：灵草有 shelflife 半衰期——采了尽快用，囤积会变质

#### C. 种田线（灵田）

- [ ] **灵田开垦**（plan-lingtian-v1）：
  - 选址：灵气 >0.3 区域（灵气太低种不活）
  - 开垦工具：采药刀 / 草镰 → 翻土
  - 补灵：骨币粉末或灵矿碎屑撒入土壤（消耗品）
  - 种植：灵草种子/根茎插入
- [ ] **成长周期**：
  - 不同灵草成长时间 30 分钟到数小时
  - 夏冬汐转影响生长速率（§十七）
  - 虫害/病害（概率事件，需手动除虫/施肥）
- [ ] **灵田风险**：
  - 长期在同一处种田 → 灵物密度上升 → 触发天道注视 → 灵气归零或刷道伥（§八）
  - 需要分散种植（"天道忌满"的物化实现）
- [ ] **种田 → 炼丹联动**：种出来的灵草直接用于炼丹——这是最自给自足的经济循环

#### D. 阵法线（地师流）

- [ ] **阵法入门**（plan-zhenfa-v1）：
  - 第一块阵石：在方块上用真元"刻"镜印（消耗真元）
  - 最简单的诡雷：一块石头埋在地下 → 5 点真元封入 → 敌人踩中引爆
  - 30 分钟内朽坏（载体没有心镜活性）
- [ ] **灵龛防御阵**（plan-niche-defense-v1）：
  - 围绕灵龛布 3-5 块联动防御阵
  - 敌人靠近 → 真元在地面形成短暂减速/伤害
  - 配合灰烬蛛（生态陷阱 + 人造陷阱叠加）
- [ ] **聚灵阵**（固元期解锁）：加速修炼——但会触发天道密度阈值

#### E. 搜打撤线（坍缩渊前奏）

- [ ] **荒野搜打撤进阶**：
  - 学会读地图：知道哪些区域灵气浓度高/低、哪些路线安全
  - 学会"在贫瘠区甩掉高手"（高手不愿在贫瘠区消耗真元追你）
  - 学会留陷阱/留假痕迹（灰烬蛛、伪灵皮蜕壳）
- [ ] **第一次伪灵脉遭遇**：
  - 天道在荒野临时升起 0.6 灵气绿洲（持续 30 分钟）
  - 玩家被吸引过去 → 可能遇到其他修士 → 自相残杀
  - narraion："又一群人聚在那片绿光里。上一次也是这样——上一次没有人活着走出来。"
  - 这是搜打撤的高级形态：**诱饵识别**（天道的陷阱 vs 真正的资源）

### 4.3 第一次流派选择（非永久，但形成倾向）

- [ ] **攻击流选择**（世界观 §五 攻击四流）：
  - **体修 / 爆脉流**：强化肉体经脉韧性，零距离贴脸强灌真元。省材料、高风险。
  - **器修 / 暗器流**：真元附着物理载体（骨刺/灵木），远程狙击。重资产、高爆发。
  - **地师 / 阵法流**：真元封入环境方块，陷阱/防御/聚灵。需要地图理解。
  - **毒蛊流**：微小飞针 + 脏真元，慢性消耗战。暴露身份 = 全服追杀。
- [ ] **防御流选择**（世界观 §五 防御三流）：
  - **截脉 / 震爆流**：皮下微型过载震爆中和入侵真元。需要精准时机判断。
  - **替尸 / 蜕壳流**：伪灵皮优先承受攻击 → 蜕落带走污染。烧材料。
  - **绝灵 / 涡流流**（固元后解锁）：掌心制造局部负灵域——抽干敌方真元暗器。高门槛、高风险残疾。
- [ ] **流派 ≠ 绑定**：玩家可以在凝脉期尝试多种打法，不锁死。但真元染色（§六）会在 ~10h 后开始形成主色——那时再换流派有代价。

### 4.4 经脉打通（第 4-6 条正经 + 奇经初触）

- [ ] **正经第 4-6 条**：~50 分钟/条
  - 此时启动"经脉路径选择"的实质体验——邻接正经的选择开始塑造打法
  - 例如：手三阴（肺/心/心包）= 偏气、利远程；手三阳（大肠/小肠/三焦）= 偏力、利近战
  - inspect UI 上可以看到自己的经脉图在形成
- [ ] **第一次被击伤经脉**：战斗中某条正经被打成 MICRO_TEAR 或 TORN
  - 该经脉流量下降 → 对应流派效果打折
  - 需要养息恢复（静坐或丹药）
  - 教学：**断了肺经的飞剑手就废了**——经脉保护的重要性
- [ ] **奇经八脉初触**（固元→通灵阶段的前奏）：
  - 此时奇经尚不可通——但 inspect UI 上已显示 8 条奇经（灰色不可选），给玩家远期的目标感

### 4.5 第一个 10 小时记忆点

- [ ] **10 小时左右的体验锚点**：
  - 经脉图已初具形态（6-8 条正经通）
  - 至少尝试过 2 种战斗流派
  - 在至少 2 个区域建立了活动范围（知道哪里有灵草、哪里有危险）
  - 灵龛存有一定物资（几枚骨币、几株灵草、一把附灵武器或几根骨刺）
  - 死过 0-3 次（取决于运气和谨慎度）
  - 至少见过一次天道伪灵脉陷阱——学会了"不是所有绿光都是资源"

## §5 P4 · 深耕（11.5h → 26.5h）— "投入"

**目标**：固元期。玩家的打法开始固化、真元染色成形、第一次面对真正的高阶威胁（坍缩渊、天劫）、打通第 7-12 条正经和奇经前 4 条。

### 5.1 凝脉 → 固元 突破

- [ ] **突破条件**：12 正经全通 + 浓度 >0.8 灵气环境静坐凝核（灵眼最佳）
- [ ] **灵眼寻找**（plan-spirit-eye-v1）：
  - 灵眼是稀有事件——随机在灵气浓度高区域生成，持续时间有限
  - 找到灵眼 = 突破固元的最优解
  - 灵眼坐标是最高价值信息（世界观 §九：信息比装备值钱）
  - 可能需要在灵眼处和其他玩家争夺——PVP 的高压点
- [ ] **固元突破**：
  - 成功：真元上限 80 → 150、可感知其他修士大致境界、固态真元核形成
  - 失败：真元核崩散 → 真元上限临时 -30、需要重新积蓄
  - narration："你体内凝出了一颗核。硬硬的，丑丑的——但至少是它让你和引气期的杂兵有了区别。"

### 5.2 真元染色成形

- [ ] **染色触发**（~10h 累计修习时长）：
  - 玩家的真元开始呈现颜色+质感
  - inspact UI 上可见：剑修 = 银白细丝、体修 = 古铜浓浆、丹师 = 药液淡黄、器修 = 凝实薄膜……
  - 染色是**长期身体改造**，不是 tag——它影响战斗效果、被其他玩家可见（信息差）
- [ ] **染色选择的压力**：
  - 到了固元期，染色已经形成——回头洗色需要在中性环境静坐慢慢中和（耗时）
  - 这时候"换流派"的成本开始变高
  - 染色和当前打法的匹配度直接决定中期效率
- [ ] **混元色可行性**（仅对刻意均衡修炼者）：
  - 任何单一修习不超过总时长 25% → 混元色
  - 效果打折（60%）、可切换、不被克制——但修炼总效率永久 -20%
  - 极少数玩家会走这条路

### 5.3 坍缩渊初探（搜打撤的终极形态）

- [ ] **第一次进入活坍缩渊**：
  - 发现裂缝：感知负压异常（固元期可感知——inspect 或 HUD 提示"前方有灵压异常"）
  - 入场规则：只能带凡物进——灵草/骨币/附灵武器在入口被抽走真元（世界观 §十六.四）
  - 标准装备：凡铁剑 + 干灵草 + 退活骨壳
- [ ] **坍缩渊内部体验**：
  - 真元条 = 秒表——盯着它下降，自己判断该走还是再深一层
  - 没有 UI 倒计时、没有红色警报
  - 浅层（-0.3）：温和，但 loot 几乎为零（历代早被搜刮尽）
  - 中层（-0.6~-0.8）：开始吃紧，偶见残卷和轻型法器
- [ ] **第一次搜刮容器**：
  - 干尸（3-5 秒搜刮）→ 凡铁、干灵草、退活骨币
  - 骨架 → 碎骨币、磨损装备
  - 搜刮时真元消耗加速 1.5 倍
- [ ] **PVP 压力**：
  - 坍缩渊内匿名加强（负压干扰神识）
  - 可能和完全陌生的人擦肩而过
  - 浅层可能有高阶修士"守门"——低阶搬运工的出关噩梦
- [ ] **第一次撤离**：
  - 站在主裂缝完成 7-10 秒撤离仪式（不能动、不能被打断）
  - 撤离中真元仍在消耗——倒计时未结束就干尸化
  - 撤离成功：所有 loot 带出 → 坍缩渊的"搜打撤"比荒野版本刺激 10 倍
- [ ] **第一次在里面死亡**：
  - 本次秘境所得 100% 掉落 + 遗骸干尸化 → 可能变道伥（世界观 §十六.六）
  - 这次死亡不是失败——遗念会告诉玩家秘境内更深层的 loot 位置
  - "坍缩渊的学费"——每个老手都交过

### 5.4 异变兽狩猎（固元→通灵 必需品）

- [ ] **异变缝合兽**（世界观 §七）：
  - 不是普通野兽——几只野兽在饥饿下互相吞噬缝合而成
  - 外貌可识别：猪的身体+狼的头+蜘蛛的腿
  - 特殊能力「灵压狂暴」：血量越低，对周围灵气的吸收越快——拖久了周围变负灵域
  - 击杀掉落：**兽核**（固元→通灵突破必需，极稀）+ 异兽骨骼（高价值）
- [ ] **兽核吸收**：
  - 击杀后在兽尸旁吸收 → 触发强烈幻觉（UI/信息干扰：画面扭曲、假 HUD 数值、假玩家名字）
  - 这是固元→通灵共鸣事件的触发条件之一
- [ ] **遭遇道伥**：坍缩渊外也会遇到（被挤到主世界的道伥）
  - 模仿玩家日常行为——远处看就是一个人在砍树/挖矿/蹲伏
  - 背对它或真元低于 20% → 瞬间切换为上古术法绝杀
  - 击杀掉落**残卷**和**破碎法宝**

### 5.5 第一次天劫

- [ ] **日常小天劫**（plan-tribulation-v1）：
  - 固元期开始，天道会不定期降下小天劫（频率随境界上升）
  - 触发条件：高消耗行为（频繁施法、大量炼丹、在灵气密集区久留）
  - 表现形式：落雷 + 灵气剧烈波动 + 修为短暂压制
  - 扛过：nothing happens（日常消耗）
  - 扛不过：受伤/真元大损
- [ ] **天劫的季节性**（§十七）：
  - 夏季天劫最可预期（雷信号清晰）——高阶修士专挑夏季渡劫
  - 冬季天劫失败率上升（灵气"凝固化"，爆发不足）
  - 汐转期天劫风险最高（天道情绪紊乱）

### 5.6 经济与社交深化

- [ ] **骨币制作**（plan-fauna-v1 的 P1）：异兽骨骼 + 真元注入 + 阵法封死 = 封灵骨币
  - 骨币里真元会缓慢流失（半衰期）——逼流转
  - 拿到骨币第一时间花掉——囤积 = 贬值
- [ ] **盲盒死信箱**（世界观 §九）：
  - 在荒野埋上锁储物箱，设置交换条件
  - 世界频道散播坐标
  - 防黑吃黑机制（箱子遇非对应破坏 → 物品化灰 + 毒气雷）
- [ ] **情报交易**：
  - NPC「食腐者」出售"谁死在哪"的坐标（可能同时卖给三波人，就为了看互杀）
  - 灵眼坐标 / 坍缩渊裂缝位置 / 安全闭关点 → 比骨币值钱

### 5.7 经脉打通（第 7-12 条正经 + 奇经前 4 条）

- [ ] **正经第 7-12 条**：~75 分钟/条（递增困难）
  - 越后面越贵——真元需求指数上升
  - 12 正经全通 → 固元→通灵突破条件之一
- [ ] **奇经初通**：
  - 奇经第 1-4 条：~3.5 小时/条
  - 奇经比正经贵一个数量级——这是"中期 vs 后期"的物理分界线
  - 通灵后解锁神识感知（可感知天道注意力/危机预警）

### 5.8 25 小时记忆点

- [ ] **固元期结束时的典型玩家画像**：
  - 12 正经全通、奇经初触 2-3 条
  - 主染色色已成形（颜色+质感可见）
  - 至少 3-5 次坍缩渊探索经历
  - 至少击杀过 1 只异变缝合兽、获得过 1-2 颗兽核
  - 至少扛过 1 次天劫
  - 灵龛升过级（防御阵+分散存储）
  - 骨币经济有感知——知道什么时候花、什么时候换
  - 地图记忆成型——哪条路安全、哪个区域有什么、哪个 NPC 可信/不可信
  - 社交关系：可能有 1-2 个长期同行者或死仇

## §6 P5 · 出头（26.5h → 51.5h）— "你不是蝼蚁了"

**目标**：通灵期。玩家开始感知天道注意力、天劫频率激增、坍缩渊深层探索、上古遗物狩猎、全服知名度上升。

### 6.1 固元 → 通灵 突破

- [ ] **突破条件**：奇经八脉通 4 条 + 一次"灵脉与天地共鸣"事件
  - 共鸣触发三选一：
    1. 击杀异变兽后于尸旁吸收灵气（最可控）
    2. 在浓度突变界面（馈赠区/负灵域交界）静坐（需要找到边界）
    3. 主动引一次小型天劫并扛过（最危险、最硬核）
  - 辅助：兽核加速共鸣
- [ ] **通灵突破**：
  - 成功：真元上限 150 → 300、可感知天道注意力（危机预警）、奇经初启（与天地共鸣）
  - 天道反应："天地一震。某个蝼蚁不配地摸到了天道的边缘。"
  - 全服不广播（通灵不如化虚那样影响全局）
- [ ] **灵脉与天地共鸣**：突破后玩家的经脉和天地产生共振
  - 世界感知范围显著扩大
  - 对灵压变化更敏感——坍缩渊入口、伪灵脉真假更容易辨识
  - 高境界维护成本启动——需要持续消耗灵气养脉，否则萎缩降境

### 6.2 坍缩渊深层（通灵期的核心内容）

- [ ] **深层进入**：
  - 深层（-0.9~-1.2）：上古法宝、道统遗物、异兽核集中出现
  - 但深层是高阶的禁区——通灵/化虚的大真元池遇高灵压差 → 数秒即被秒
  - 反而是低阶（凝脉/固元）能在深层苟 20-30 分钟
  - → 搬运工经济生效：低阶挖宝、高阶守浅层收割
- [ ] **通灵期在坍缩渊中的定位**：
  - 中层还能舒服活动、深层只能"速降——拿一件就跑"
  - 决策：自己下去拿？还是守浅层收过路税？
  - 坍缩渊 fighting 比外面更苛刻——每次施法双重消耗（己方真元 + 环境负压）
- [ ] **上古遗物获取**（坍缩渊内 1% loot）：
  - 传说档容器（法阵核心/坛柱）→ 搜刮 25-40 秒
  - 上古遗物属性：极高（一击破护体真气、一卷开 12 正经）、极脆（1-3 次使用即碎）
  - 取出 = 拔坍缩渊骨架 → 灵压深化、塌缩加速
  - 最后一个被取走 → 全坍缩渊塌缩 → race-out（3 秒塌缩裂口）
- [ ] **race-out 体验**：
  - 塌缩瞬间——负压翻倍、随机开 3-5 个塌缩裂口（3 秒撤离时间）
  - 所有还在里面的人拼命往裂口跑
  - "你以为你拿到了最后一件遗物？现在你得活着把它带出去。"
  - 慢了 → 随坍缩渊一起化为死域（真正终结）

### 6.3 天劫加剧

- [ ] **天劫频率**：通灵期天劫频率显著上升（天道主动针对高消耗修士）
  - 每次天劫都是一次小型"渡劫"——架势、时机和准备都很重要
  - 可以躲入负灵域逃避天劫（天道在负数区无法索敌）→ 代价是跌落境界
- [ ] **渡虚劫前奏**：
  - 通灵→化虚需要渡虚劫（专属于通灵修士的天劫，强度极高）
  - 全服广播："有人在渡虚劫。"
  - 所有人都知道位置 → 可以来截胡
  - 活下来就突破，死了回到通灵初期

### 6.4 全服知名度

- [ ] **通灵期 ≈ "出头"**：
  - 全服通灵修士不超过 10-20 人
  - 固元+可感知其他修士境界 → 看到你时显示"此人气息浩瀚"
  - 名字开始被其他玩家记住——既是荣耀也是靶子
- [ ] **社交变局**：
  - NPC「食腐者」更喜欢卖你的坐标（高价值目标）
  - 散修 NPC 见到你 → 丢下低级资源疯狂逃窜
  - 低阶玩家可能组队来"猎通灵"（负压战术：诱入负灵域反杀）
- [ ] **天道关注**：
  - 天道 narration 中可能出现你的名字/特征
  - 劫气标记触发率更高
  - 灵物密度阈值对你的基地更敏感

### 6.5 顿悟的高光时刻

- [ ] **可能在此阶段触发的高价值顿悟**（世界观 §六.三）：
  - 首次扛过天劫 → 天劫相关顿悟（雷抗/天象感知/劫气削弱）
  - 在负灵域不死生还 → 负压适应顿悟
  - 目睹他人渡虚劫 → 突破灵感顿悟
  - 某门功法练满 → 功法终极顿悟
- [ ] **顿悟选择不可更改**：这是角色一生的关键决断点

### 6.6 经脉打通（奇经后 4 条 + 冲击化虚）

- [ ] **奇经第 5-8 条**：~5 小时/条
  - 奇经全通 → 通灵→化虚突破条件
  - 任督二脉（统御属性）：调和真元，开通后大幅提升真元上限
- [ ] **化虚前夜**：
  - 当新一条奇经打通后，inspect 上只剩"渡虚劫"的提示
  - 玩家感知到天道注意力急剧上升——"它知道你在准备什么。"
  - 开始囤资源、找安全渡劫点、或寻人护法

## §7 P6 · 天花板（51.5h+）— "化虚之后"

**目标**：化虚境。天花板体验——全服只有 1-2 人能达到的境界。寿元管理、天道最大敌意、终局内容。

### 7.1 渡虚劫

- [ ] **渡虚劫流程**（plan-tribulation-v1）：
  - 触发：玩家主动引渡虚劫（奇经全通后可选）
  - 天道降下专属天劫——强度极高、时间较长（~5-10 分钟）
  - 全服广播（匿名或具名？取决于 §十一 规则）
  - 其他玩家可见天劫气象（全服天空变色、落雷方向可辨识）→ 可以循迹而来
- [ ] **渡虚劫中**：
  - 分段式天劫——每段强度递增
  - 其他玩家可以趁机偷袭（你最脆弱的时候）
  - 需要护法者或选址极度隐蔽
- [ ] **结果**：
  - 成功 → 化虚境达成：真元上限 300 → 500、奇经八脉全通、服务器内仅容 1-2 人
  - 天道："天地又多了一个不该有的存在。"
  - 失败 → 回到通灵初期、经脉受损、需要长时间恢复
  - 渡虚劫失败 ≠ 角色终结（除非运数/劫数用完）

### 7.2 化虚境体验

- [ ] **化虚境特殊性**：
  - 全服仅容 1-2 人（天道不允许更多——消耗太多灵气）
  - 如果已有 2 个化虚 → 第三个通灵无法触发渡虚劫（天道封锁）
  - 化虚修士的存在本身就是"对其他通灵者的障碍"
- [ ] **天道敌意最大化**：
  - 天劫频率极高（维护成本巨大）
  - 天道 narration 中频繁提及——"化虚者的每一步，天地都在计数。"
  - 灵物密度阈值对化虚者的基地极度敏感
- [ ] **寿元管理**（世界观 §十二）：
  - 化虚寿元上限 2000 年（但每次死亡扣 100 年）
  - 在化虚境死亡 → 回到通灵初期（境界 -1）+ 扣寿 100 年
  - 这是最严酷的"高处不胜寒"
- [ ] **负灵域极度危险**：
  - 化虚的大真元池遇深负压 → 秒级被抽干（世界观 §二 "境界越高抽吸越恐怖"）
  - 浅层坍缩渊也要速进速出
  - "你能扛天劫，但扛不住物理法则。"

### 7.3 终局内容

- [ ] **化虚境的"游戏目标"变化**：
  - 不再以突破为驱动（没有更高境界）
  - 改为：寿元最大化、道统传承、影响世界格局
- [ ] **道统传承**：
  - 化虚修士的功法残篇、法宝 → 死亡后成为其他人的"道统遗物"
  - 可通过 NPC 或死信箱传给指定继承人
  - 一生记录（生平卷）入「亡者博物馆」永久公开 = 名留青史
- [ ] **世界影响**：
  - 化虚修士的行为能影响大片区域的灵气分布
  - 可以"镇压"某个坍缩渊（延缓塌缩）或"引爆"某个区域
  - 天道 narraion 中化虚修士成为世界的"变量"

### 7.4 多周目与传承

- [ ] **角色终结后**（不管什么境界）：
  - 一生记录入馆（亡者博物馆）→ 新角色可查阅前人路径
  - 道统遗物掉落 → 可在世界中被拾取
  - 玩家开新角色：从醒灵重新开始，但**带着所有知识**（脑内经验）
- [ ] **多周目差异**：
  - 第一次玩：~100h 到化虚，运数充足但什么都不知道
  - 第二次玩：~60-80h 到化虚（知道路线），运数告急（前角色已经消耗），概率 5%
  - 第十次玩：20h 就到凝脉（熟练工），但每次死都可能直接终结（概率 5%）
  - 每个新角色继承前人的"知识"（玩家脑内），不继承"实力"（机制隔离）

## §12 未解决问题（Open Questions）

1. **前 30 分钟是否需要显式 UI 教程？** 天道 narration 是"冷漠的古意旁白"，不适合当教程。是否需要一个"残土旅人笔记"（类似《只狼》的教程弹窗，但世界观内）？还是完全靠环境线索隐式引导？
2. **"同玩家多角色"的多周目加速**：plan 目前没有定义同玩家开新角色时的福利（跳过教学、初始物资？还是完全从零？）
3. **化虚期的"服务器级影响"**：化虚修士改变区域灵气/镇压坍缩渊——这些操作需要 server 端暴露什么样的 GM 接口？
4. **坍缩渊"搬运工经济"的博弈平衡**：低阶拾荒但被高阶在浅层收割——这个循环如何确保低阶也有机会出货而不至于永远当炮灰？

6. **narration 的触发密度**：多高频率的 narration 算"太多"（烦人）？多低算"太少"（天道死了）？
7. **灵田"天道注视"的具体触发阈值**：一个玩家几块田、连续种多久触发灵气归零？阈值是否需要可配？
8. **汐转期的"随机劫气标记"如何通知玩家**？HUD icon ？narration？还是玩家自己观察到运气变差才后知后觉？
