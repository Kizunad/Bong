# Bong · plan-zhenfa-v1 · 模板

**阵法专项**。HUD §8.2 提到 "Dynamic XML Screen 阵法布置"，combat 引用 Zhenfa 技能，本 plan 定义：布阵 / 触发 / 破阵 / 阵法种类。

**世界观锚点**：`worldview.md §十三` 明确"中途没有传送、没有坐骑"——**禁止传送阵**。`worldview.md §七` "灵物密度阈值"——**聚灵阵会被天道盯上**，不是无脑铺。`worldview.md §七` 的「欺天阵」是已确立的核心阵法范例（向天道广播假劫气权重）。

**交叉引用**：`plan-combat-v1.md` · `plan-client.md`（DynamicXmlScreen）· `plan-worldgen-v3.1.md` · `worldview.md §七/§十三`。

## §1 阵法种类

| 类型 | 用途 | 范围 | 代表 / 备注 |
|---|---|---|---|
| 聚灵阵 | 加速修炼 | 小 | 触发天道密度阈值 → 灵气归零或刷道伥（worldview §七） |
| 防御阵 | 护山 / 护身 / 护灵龛 | 中/大 | 与灵龛保护配合 |
| 幻阵 | 迷惑入侵者 / 隐藏阵眼 | 中 | 配合匿名系统 |
| 杀阵 | 陷阱 / 死信箱自爆 | 小/中 | 见 worldview §九 盲盒死信箱 |
| 欺天阵 | 向天道广播假劫气权重 | 远点（500 格外） | worldview §七 line 494 核心案例 |
