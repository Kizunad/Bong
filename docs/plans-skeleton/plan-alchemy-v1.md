# Bong · plan-alchemy-v1 · 模板

**炼丹 / 炼器专项**。HUD §10 快捷使用栏全是丹药，inventory 里 Consumable/Weapon 来源未定；本 plan 定义产出流程。

**世界观锚点**：`worldview.md §九/§十` 资源体系明确末法时代**没有"仙器"**——飞升者属上古遗产，当代炼器最高只到稀有载体材料级。本 plan 不得引入"仙器"品阶。`worldview.md §三` "材料/丹药永远是辅助"——丹药给加速/保险，不是必需，**不做"极品丹无副作用"**。

**交叉引用**：`plan-inventory-v1.md` · `plan-cultivation-v1.md` · `plan-HUD-v1.md §10` · `worldview.md §三/§九/§十`。

---

## §0 设计轴心

- [ ] 炼丹 / 炼器是玩家产出 Consumable/Weapon 的主要途径
- [ ] 非纯数值合成：有失败 / 品阶 / 丹毒 / 丹炉爆裂代价
- [ ] 与修炼境界挂钩（炉火 = 真元投入）
- [ ] 玩家可调火候（玩法深度）
- [ ] **万物皆有成本**——不存在"无副作用极品"

## §1 炼丹流程

```
选方 → 备料 → 起炉 → 火候调节（交互） → 成丹 / 炸炉
```

- [ ] 交互 UI（DynamicXmlScreen 或 BaseOwoScreen）
- [ ] 火候参数（温度 / 时长 / 真元投入）
- [ ] 成功率公式（境界 + 配方熟练度 + 区域灵气）

## §2 丹药品阶（皆有代价）

| 品阶 | 效果倍率 | 丹毒 / 副作用 |
|---|---|---|
| 下品 | 0.6 | 高丹毒 |
| 中品 | 1.0 | 中丹毒 |
| 上品 | 1.5 | 低丹毒（仍有） |
| 极品 | 2.0 | 低丹毒 + 罕见后遗（如真元上限暂时回调） |

## §3 炼器流程

- [ ] 材料 → 坯 → 淬炼 → 铭文 → 开光
- [ ] 品阶：**凡器 / 法器 / 灵器 / 道器**（无"仙器"）
- [ ] 道器对应 worldview §九 "顶级资产" 等级，需异变兽核 + 残卷工艺
- [ ] ForgeWeapon 技能（combat §5 钩子）
