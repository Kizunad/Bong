# Bong · plan-social-v1 · 模板

**社交 / 关系 / 声名专项**。HUD social channel 只给聊天，本 plan 定义匿名社会下的稀缺关系结构。

**世界观锚点**：`worldview.md §一` line 4 "**没有宗门收你**"；`worldview.md §十一` "玩家之间默认敌对的陌生人"、修士之间**默认不显示名字**。本 plan **不得引入"正道/魔道阵营"、"掌门/长老/内门/外门弟子"等传统宗门层级**——末法残土是匿名利己的原子化社会。

**交叉引用**：`plan-npc-ai-v1.md` · `plan-agent-v2.md` · `plan-HUD-v1.md §2.3` · `worldview.md §一/§十一/§十二`。

---

## §0 设计轴心

- [ ] 默认匿名 + 默认敌对（worldview line 587, 599）
- [ ] 关系是稀缺品，建立成本高、暴露代价高
- [ ] 不做 MMO 公会 / 不做正魔阵营
- [ ] 师承可遇不可求（NPC 散修偶尔传一招，非系统化收徒流程）

## §1 匿名与暴露

- [ ] 默认不显示名字（外观仅暗示境界段）
- [ ] 暴露名字的四种触发：聊天 / 交易 / 被天道点名 / 死亡（worldview line 606-610）
- [ ] 固元+ 可看到对方大致境界段

## §2 关系类型（稀疏图，非组织架构）

| 类型 | 建立方式 | 数据结构 |
|---|---|---|
| 师承（偶发） | NPC 主动收 / 玩家求教成功 | 单向有向边 |
| 同行者 | 长期相处的标记 | 双向边 |
| 盟约 | 显式立誓（生平卷可查） | 双向边 + 条款 |
| 死仇 | 杀过我 / 我杀过他 | 双向边 + 时间戳 |

**不做**：固定门派/帮会/公会/总掌门体系。

## §3 声名（不是阵营声望）

- [ ] 个人「名号」由行为累积（NPC 口耳相传 + 生平卷公开片段）
- [ ] 没有"正道声望/魔道声望"二元轴
- [ ] 染色谱（worldview §六）才是真正的"立场"标签——你的真元颜色 = 你修了什么

## §4 NPC 散修态度

- [ ] 默认警惕（worldview §十一 危机分层）
- [ ] 受声名 / 染色 / 当前业力影响
- [ ] agent 推演 NPC 派系小冲突（但不形成长期阵营）

## §5 玩家 ↔ 玩家

- [ ] 切磋 / 交易（暴露身份）
- [ ] 盟约/死仇标记进生平卷
- [ ] 不强制 PK，但不做"安全区"概念（灵龛是私有点不是公共安全区）

## §6 数据契约

- [ ] RelationshipStore（稀疏图）/ ReputationStore（个人，非阵营）
- [ ] Channel
