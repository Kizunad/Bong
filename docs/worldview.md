# 世界观 — 末法残土

> "天地不仁，灵气已薄如纸。你醒来时，身处一片灵脉将枯的荒土。
> 这里没有宗门收你，没有前辈指路，连一株像样的灵草都要和野兽抢。
> 你唯一的优势是——你会死很多次，而每一次死亡都不是白死的。"

---

## 一、世界基调

### 末法时代

这不是灵气充沛的上古盛世，而是一个**灵脉接近枯竭的末法残土**。

曾经的大能们早已飞升或陨落，留下的只有残破的洞府、失控的禁制、和一片被过度汲取后濒临崩溃的灵脉网络。天道不再庇佑众生——它只是一台冷漠的平衡机器，在灵气总量持续下降的世界里，执行着越来越残酷的"优化淘汰"。

**核心设定**：
- 全服灵气总量是**恒定且缓慢衰减**的，不会凭空产生
- 你修炼消耗的灵气，就是别人少掉的灵气
- 天道的职责不是帮你成长，而是**确保这个将死的世界不会死得太快**
- 当某个区域灵气被吸干，那里会变成死域——连野兽都活不了

### 基调关键词

**匮乏、算计、信息差、搜打撒、苟得住才活得久、但苟太久会被天道盯上**

## 二、修炼体系

### 境界

不叫"练气筑基金丹元婴"，太传统了。末法时代的修士不配用上古称呼。

| 阶段 | 名称 | 本质 | 真元上限 | 世界感知 |
|------|------|------|---------|---------|
| 0 | **醒灵** | 凡人刚感知到灵气 | 10 | 能看到灵气浓淡（HUD 灵气条） |
| 1 | **引气** | 能主动吸纳灵气 | 30 | 能感知 50 格内生物气息 |
| 2 | **凝脉** | 体内形成灵脉雏形 | 80 | 能感知区域灵气精确值 |
| 3 | **固元** | 灵脉稳固，可施展术法 | 150 | 能感知其他修士的大致境界 |
| 4 | **通灵** | 灵脉与天地共鸣 | 300 | 能感知天道注意力（危机预警） |
| 5 | **化虚** | 半步超脱 | 500 | 服务器内仅容 1-2 人（天道不允许更多） |

**关键规则**：
- 境界**可以掉落**。真元长期不足（低于上限 20%）→ 灵脉萎缩 → 降一阶
- 突破不是积攒够了就自动升，需要**特定资源 + 特定条件**（见下文）
- 高境界的维护成本极高——你需要持续消耗灵气来"养脉"，否则掉回去
- 化虚境是天花板。天道会主动针对化虚修士——因为它们消耗太多灵气

### 突破条件（不是打怪升级）

```
醒灵 → 引气：
  找到一处灵气浓度 > 0.5 的地点，静坐 3 分钟不被打断
  （听起来简单？灵气 > 0.5 的地点往往有其他修士/野兽盘踞）

引气 → 凝脉：
  消耗「凝脉散」×1（需要 3 种灵草炼制，配方不固定——天道会变）
  + 真元池满 + 一次性消耗 20 真元

凝脉 → 固元：
  找到一处「灵眼」（极稀有地点，全服同时最多 3 个）
  在灵眼处打坐 5 分钟不被打断
  + 消耗「固元丹」×1

固元 → 通灵：
  需要击杀一只「异变兽」（天道随机刷新的强化怪）
  + 在其尸体旁吸收灵气突破
  + 真元池满

通灵 → 化虚：
  天道降下「渡虚劫」——专属天劫，强度极高
  活下来就突破，死了就回到通灵初期
  全服广播："有人在渡虚劫。"（所有人都知道你的位置）
```

### 真元（不是蓝条，是命）

真元不只是施法资源，它是**你的命脉**：
- 真元 = 0 → 不会立刻死，但灵脉开始崩溃 → 10 分钟内不补充就降境界
- 真元回复**只能靠灵气浓度高的地方静坐**或使用消耗品，没有自动回复
- 被攻击会同时掉血和掉真元（修士之间的战斗是在消耗对方的"底蕴"）
- 你打赢了，但真元耗尽、满身伤 → 你赢了战斗但输了生存

## 三、资源与匮乏

### 灵气是零和的

全服灵气总量固定（SPIRIT_QI_TOTAL = 100），且天道会缓慢回收（每个时代衰减 1-3%）。

当你在某个区域修炼，你在**消耗那个区域的灵气**。修炼完走了，那块地就变贫瘠了。这意味着：
- 新手区的灵气会被先来的人吸干
- 好地方是**会被用完的**，然后变成废地
- 天道会缓慢重分配灵气（从死域向无人区转移），但速度很慢
- 高手聚集的区域灵气消耗最快——这就是天道要"驱散强者"的物理原因

### 资源种类

| 资源 | 获取方式 | 稀缺度 | 用途 |
|------|---------|--------|------|
| 灵草 | 野外采集，区域灵气 > 0.3 才会生长 | 中等 | 炼丹基础材料 |
| 矿石 | 挖掘特定方块（矿脉有限，挖完就没了） | 高 | 炼器、交易硬通货 |
| 丹药 | 灵草 + 配方炼制 | 高 | 突破、回复、增益 |
| 灵眼坐标 | 探索发现（位置随天道变化） | 极稀 | 凝脉→固元突破必需 |
| 残卷 | 遗迹/洞府探索 | 极稀 | 学习术法（战斗能力核心） |
| 异变兽核 | 击杀异变兽 | 极稀 | 固元→通灵突破必需 |

### "搜打撤"循环

这是核心游戏循环：

```
搜 → 探索未知区域，寻找灵草/矿脉/遗迹
     风险：野兽、其他修士、灵气耗尽
     信息差：你发现了一个灵眼，别人不知道——这就是你的优势

打 → 遇到资源点，要么和野兽/NPC 抢，要么和其他玩家抢
     战斗消耗真元，赢了也可能空虚
     要判断"这场仗值不值得打"

撤 → 拿到东西就走，回到安全点修养
     知道什么时候该跑比知道怎么打更重要
     被追杀时要利用地形/灵气贫瘠区甩掉追兵
     （高手不愿在贫瘠区追你——浪费真元）
```

## 四、安全与危机

### 安全空间（苟命点）

每个玩家出生时获得一个**「灵龛」**——一个只有自己知道坐标的隐蔽点。

灵龛规则：
- 可以在任意位置设置（消耗一次性道具「龛石」）
- 灵龛方圆 5 格内 NPC 不会主动攻击、其他玩家无法破坏方块
- 但灵龛**不提供灵气**——你不能在里面修炼，只能藏东西和养伤
- 灵龛坐标如果被其他玩家发现（通过追踪/推理），保护失效
- 死亡后复活在灵龛位置（而非世界出生点）

**设计意图**：给玩家一个"家"的安全感，但这个家很脆弱。你得经营好你的秘密。

### 匿名系统

修士之间**默认不显示名字**。

你看到的其他玩家是：
- 一个穿着普通的人形（外观由境界段位决定，但不精确——引气和凝脉看起来差不多）
- 头顶没有名字
- 如果你的感知力够强（固元+），可以看到对方**大致的境界段**（"此人气息在你之上"/"此人气息微弱"）

**暴露名字的方式**：
- 主动交流（向对方发送聊天消息会暴露自己名字）
- 交易（双方互相可见）
- 被天道点名（天劫/narration 提到你时，全服知道）
- 死亡时（杀手会看到死者名字，但旁观者不一定看到）

**设计意图**：每个陌生人都是潜在威胁。你不知道对面是新手还是高手。这就是危机感的来源之一。

### 危机来源分层

```
弱者（醒灵/引气）的危机：
  - NPC 野兽（对你有实质威胁）
  - 其他玩家的随手一击（你不值得被认真对待，但也不值得被放过）
  - 灵气不足，找不到能修炼的地方
  天道态度：偶尔怜悯，投放一点小机缘（灵草刷新在你附近）

中等（凝脉/固元）的危机：
  - 其他同阶玩家的资源竞争
  - 异变兽（天道刷新的强化怪，你想突破就必须打）
  - 突破失败的代价（消耗了资源但没成功 → 白亏）
  天道态度：中立，偶尔制造"赶路"事件（你所在区域灵气骤降，逼你迁移）

强者（通灵/化虚）的危机：
  - 天劫频率显著上升（天道主动针对你）
  - 真元维护成本极高（不修炼就掉境界，修炼就消耗大量灵气→惹眼）
  - 全服知名度上升（天道叙事会提到你："某修士已近化虚…"）→ 成为靶子
  - 化虚渡劫是全服广播 → 所有人都可以来"截胡"
  天道态度：敌意。你的存在消耗太多灵气。天道想淘汰你。
```

## 五、死亡与轮回

### 死亡不是结束，是信息

借鉴《我的模拟长生路》的核心——死亡是**学习**。

**死亡惩罚**：
- 掉落身上 50% 随机物品（掉在死亡点，任何人可以拾取）
- 境界降一小阶（引气三层 → 引气二层）
- 真元归零
- 复活在灵龛（如果有）或世界出生点

**但是——死亡给你信息**：

死亡时玩家会看到一条「遗念」（death insight），由天道生成：

```
"你死于血谷深处。临死前你感知到：
  - 此处地下 30 格有微弱灵气波动
  - 杀死你的那只异变兽，左腿有旧伤
  - 东北方向 200 格外，灵气浓度异常高"
```

这些信息可能是：
- 真实的情报（天道根据世界状态生成的真实信息）
- 部分真实（方向对但距离不准）
- 极少数情况下是误导（天道的"无常"）

**设计意图**：每次死亡都推动你对世界的认知。你死了 10 次后，你比新来的人知道更多——这就是你的**信息差优势**。鼓励玩家记录、推理、规划。

### 「模拟」感

虽然不像小说里有字面意义的"模拟"，但体验上接近：
- 第一次进入：什么都不知道，死得很快
- 第三次进入：知道哪里安全、哪里有资源、哪个方向别去
- 第十次进入：你已经有了心中的"地图"，知道什么时候该冒险
- 这种**经验积累不在角色身上，在玩家脑子里**

## 六、天道行为准则

天道不是帮你的，也不是害你的。它是这个衰败世界的**免疫系统**。

### 天道的唯一目标

**让这个世界多活一天。**

灵气在衰减，修士在消耗。天道的所有行为都服务于一个目标：减缓灵气消耗速度，延长世界寿命。

### 天道的手段

```
温和手段（灵气微调）：
  → 把灵气从过度聚集的区域匀给贫瘠区
  → 在无人区慢慢恢复灵气
  → "移风换水，不动声色"

中等手段（制造压力）：
  → 对高消耗修士所在区域降低灵气
  → 在强者区域刷新异变兽（既是威胁也是诱饵）
  → 发布天象预兆让修士自行迁移
  → "天象有异，智者当避"

激烈手段（天劫）：
  → 直接对高境界修士降天劫
  → 在灵气将尽的区域触发「域崩」（区域毁灭，所有人必须撤离）
  → 化虚渡劫全服广播
  → "天道不容"

隐性手段（叙事引导）：
  → 通过 narration 暗示某个方向有机缘（把玩家从过度聚集区引开）
  → 暗示某个修士"气运将尽"（让其他玩家去当天道的刀）
  → "冥冥之中自有天意"——天道利用玩家之间的贪欲来平衡
```

### 天道叙事的语调

不是温暖的旁白，不是游戏提示。是**冷漠的、有古意的、偶尔带嘲讽的**。

```
好的叙事：
  "血谷灵脉又枯了三分。仍有蠢人在那里打坐。"
  "此间有修士渡劫。天地为之色变。旁观者…自求多福。"
  "东荒深处传来一声低吼。是灵兽？是机缘？是坟墓？"

不好的叙事（太温暖/太游戏化）：
  "注意！血谷灵气下降了！"
  "恭喜！你发现了一个灵眼！"
  "小心，前方有危险的怪物！"
```

## 七、世界地理草案

### 尺度感

MC 里 1 格 ≈ 1 米。这个世界要有**荒野求生**的距离感：

- 区域之间间隔 **2000-5000 格**，中间是大片无人荒野
- 单个区域本身面积 **1000×1000 格** 左右
- 全地图约 **20000×20000 格**（±10000），只有中心区域有预生成地形
- 外围逐渐退化为灵气近零的荒原，再远就是虚空（世界边界）
- 走一趟远路要 **10-20 分钟真实时间**，中途没有传送

这意味着"你知道灵眼在哪"是一条价值极高的信息——因为跑错方向就是 10 分钟白费，还可能在路上被截杀。

### 荒野（区域之间）

区域之间不是空白——是大面积的**灵气贫瘠荒野**（灵气 0.1-0.2）：
- 偶尔刷新低级野兽
- 零星的灵草（极少，可能跑 500 格才找到一株）
- 偶尔出现天道临时生成的「野灵脉」——一处维持 30 分钟的高灵气点（灵气 0.6），之后消散
- 荒野中遇到人的概率很低——但遇到了就很紧张，因为对方也在赶路，可能和你去同一个地方

## 八、这个世界的一天

用一个虚构的玩家视角，展示完整体验循环：

```
你第 4 次醒来。

上一次你死在血谷，被一只异变蜘蛛咬死了。但死前你感知到了——
血谷东侧有一处灵气波动，地下大约 20 格。你记住了。

这次你从灵龛醒来。引气二层。真元 0/30。
灵龛在青云残峰的一个山洞里，安全但没灵气。
你必须出去。

你沿着上次探索过的路线走向灵泉湿地。那里灵气够，可以恢复真元。
路上你看到一个人影——没有名字，看不出境界。
你选择绕路。不值得冒险。

到了湿地边缘，你找了个隐蔽的角落打坐。真元慢慢回复。
期间天道的声音在耳边响起：
"灵泉湿地之灵气，今又薄了几分。贪者众而灵脉衰。"
你知道这意味着这里的灵气在被消耗。你不能待太久。

真元回到 25/30。你起身，摘了两株灵草——够炼一份凝脉散了。
但你还差一种材料：血谷才有的「赤髓草」。

你想起上次死前的信息——血谷东侧地下有波动。也许那里有赤髓草。
但血谷很危险。上次你就死在那里。

你决定去。因为你知道了蜘蛛在哪里，可以绕开。
这就是信息差。这就是你比第一次进来时更强的原因——
不是因为等级高了，而是因为你死过，你知道了。

你进了血谷。避开了蜘蛛。找到了地下洞穴。
里面有三株赤髓草。还有一个你没见过的矿脉。

你快速采集。然后——

你听到身后有脚步声。

你没有回头，直接朝灵气最稀薄的方向跑。
因为你知道：如果对方是高手，在贫瘠区追你不划算。
如果对方是同阶，你跑得比他快——你的真元还剩 20，够你跑出去。

你跑出了血谷。对方没追上来。

回到灵龛。你有了凝脉散的所有材料。
但你不急着炼。你要先找一处灵气 > 0.5 的安全点——
因为突破需要 3 分钟不被打断。

你开始规划下一步。
```

## 九、设计哲学总结

1. **强大不是终点，是新的危险的开始。** 每升一个境界，维护成本更高，天道更敌视你。
2. **信息比装备更值钱。** 知道灵眼在哪、知道哪条路安全、知道对手是谁——这些比一把好剑重要。
3. **玩家之间是默认敌对的陌生人。** 但合作有时比对抗更理性——关键是你怎么判断。
4. **死亡是学费，不是惩罚。** 每次死都给你一条情报。老玩家强在脑子里，不只是数值上。
5. **天道不是 GM，是生态调节器。** 它不在乎你的感受，它在乎灵气总量的消耗速率。
6. **苟得住不代表能赢，但活着才有下一次机会。**
