## 0. 权威设定（云端**必须**在开工前精读并严格遵守）

### 0.1 境界体系：**六阶**，不是九阶

来源 `docs/worldview.md` L64-71，`plan-cultivation-v1.md` L132-139：

| 阶 | 名 | 代号 | 真元上限 |
|---|---|---|---|
| 0 | **醒灵** | Awaken | 10 |
| 1 | **引气** | Induce | 30 |
| 2 | **凝脉** | Condense | 80 |
| 3 | **固元** | Solidify | 150 |
| 4 | **通灵** | Spirit | 300 |
| 5 | **化虚** | Void | 500 |

化虚是天花板（worldview L80），全服只容 1-2 人。主流人群处于凝脉-固元段。**禁止**出现任何传统仙侠境界称谓。

### 0.2 战斗流派：**四攻三防七流**，每个都是**双名**

来源 worldview L204-259。写作时**必须用双名**，不要简写：

- 攻击四流：**体修 / 爆脉流**、**器修 / 暗器流**、**地师 / 阵法流**、**毒蛊流**
- 防御三流：**截脉 / 震爆流**、**替尸 / 蜕壳流**、**绝灵 / 涡流流**

**核心约束**：末法时代**不存在外放护盾**，三防均针对"已入体"的攻击（L236-252）。写"护体真气罩"违反世界观。

### 0.4 灵压三域 / 经脉拓扑

**三域**（L29-54）：
- 馈赠区（灵气 > 0）：修炼地
- 死域（灵气 = 0）：真元自然流失
- 负灵域（灵气 < 0）：天地反吸真元，境界越高吸越快；**战术价值**：通灵+可躲天劫、低阶可诱高阶进入实现反杀

**经脉**：12 正经（手足三阴三阳）+ 8 奇经（任督冲带 + 阴阳维跷）= 20 条。任督开通大幅抬升真元上限。

### 0.5 骨币经济：**半衰期指数衰减**

来源 worldview L515-537：
- 封灵骨币是唯一硬通货（修士注真元入异变兽骨制成）
- "一枚含 10 点真元的新币，箱子里放一个月可能只剩 2 点"（L520）—— 半衰期，不是线性扣减
- 真正保值：**信息**（灵眼坐标、闭关点、验证过的死亡遗念）。全世界"骨币会贬，记载不朽"（网站 tagline）
- 灵物操作磨损（L485-488）：箱子取出/背包转移扣 1%-5%，是天道的隐性交易税
- **禁止**出现"灵石"—— 唯一硬通货是骨币

### 0.6 死亡三机制

来源 worldview L640-790：
- **死亡不是终结，但终会终结**：世界观允许多次重生，但机会会缩减，最终归于"道消"。**具体次数 / 概率公式 / 触发条件属游戏内可探索知识，馆藏文本不得直接揭示数字**（玩家会在实战中得知，写出来破坏沉浸感）。可写"运数将尽"、"天道不再护佑"、"第几世已难记清"之类的氛围语，不要列表化
- **遗念**（L748-771）：死亡瞬间天道生成，境界越高越详细；"天道不屑编造谎言"
- **生平卷归档**（L694-738）：角色死透后生平被归档留存供后来者参考，是世界内的历史机制。library-web 站本身**不是**这个机制的载体，只是普通的世界观文献汇集

### 0.7 六个地理区域（含北荒）

worldview L844-851：初醒原（0,0）/ 青云残峰（-3000,-2000）/ 灵泉湿地（-2500,2500）/ 血谷（3000,-2500）/ 幽暗地穴（2000,3000）/ 北荒（0,-7000）。**没有坐骑、没有传送**（L853）。

## 1. 风格锚（直接引用基准）

云端写作时以下三条作为**语气压舱**：

1. **tagline**（`library-web/src/layouts/Base.astro:17`）："骨币会贬，记载不朽"
2. **副标题**（`pages/index.astro:21-23`）："天地灵气将尽，唯文字不灭。此间所录，皆残土之真。"
3. **页脚**（`Base.astro:49`）："凡可立卷者，皆应先入馆藏，再论买卖与实现。"

**叙事立场**：library-web 是**末法残土里流传的一部残缺丛书**——由若干作者在不同时代留下的残篇 / 笔记 / 口信汇集而成。与游戏内的"亡者博物馆"机制**无直接关联**，纯粹是世界观载体。风格是"前人遗留，供后来者阅读"的第一手文本（**不是**第三人称百科），语气陈述性冷漠，古意简省，带世界末日感但**不悲悯**。

### 1.1 文风基准：**水浒传笔法**

**核心参照**：明代白话长篇小说《水浒传》的语言层级——文绉绉与大白话之间，既有"话说"、"且说"、"看官"式的叙事调度，也有"一拳打去"、"抡起哨棒"式的直白动作描写。**典型特征**：

- **文言骨架 + 白话肌肉**：句式偏古（"但凡……者"、"今……却如何"、"莫不是……？"），但词汇多用白话，不堆艰涩字
- **三字/四字排比**：写景写人擅用短句连缀（"灵气渐薄，骨币渐贬，人心渐冷"）
- **旁白调度**：允许"话说"、"却说"、"看官须知"、"按下不表"、"诗曰"之类第二人称呼唤
- **口语感对白**：引用他人话时用直白口语（"你这厮……"、"不妨事"、"也罢"），不要全用文言
- **有诗为证**：偶尔插入短诗或联句（四句或两联即可），但不要每篇都有

**水浒传节选锚点**（撰写时自我对照）：
- "话说……那时正是盛夏……但见：……"（开头调度 + 写景）
- "鲁智深听得，跳起身来，道：'洒家不管他！'"（人物对白）
- "大树一般倒下，把那店小二吓得目瞪口呆"（动作+旁人反应）
- "有诗为证：……"（插诗收束）

**不要模仿的水浒特征**：章回体（"第一回…分解")、过长的武打段、单篇超过 1200 字的铺叙。本站馆藏每册短小精悍，不是演义。

### 1.2 文体分布（每本书指定一种）

- **散修手札**（最常见，带实地体温，第一人称 + 水浒口语）
- **宗门长老注疏**（学究、偏理论，文言骨架更硬，少口语）
- **末法遗老追忆**（缅怀灵气鼎盛，"昔年……今……"对比结构）
- **商贾农夫记账**（接地气、白话多、算账逻辑直接）
- **死者遗书 / 临终口信**（短、碎、带血气；可全用口语）
- **天道口吻摘录**（冷漠陈述式，仅少量条目；文言较重，无我称）

### 1.3 风格原句样本（worldview + 旧稿直引 + 水浒类比）

**世界观自有**：

## 4. 写作硬约束

### 4.1 禁止词

- "灵石"（唯一硬通货是骨币）
- "九阶 / 传统仙侠境界称谓"（只有六阶）
- "护体真气罩 / 外放护盾"（末法无此物）
- "传送 / 瞬移 / 坐骑"（worldview L853 明确无）
- "系统 / 面板 / 属性点 / 升级"（现代游戏词汇）
- "AI / LLM / 由天道生成 / 我是..."（破戏）

### 4.2 格式硬要求

每册 md body **600-1200 字**（不计 frontmatter），分 2-4 节。

quote 字段 **20-40 字**（超过一行就是错的）。

`shelf` 字段建议格式 `"{天干/地支/数字} {方位/类别}·{子架}"`，如 `"丙三·币轨残片"` / `"乙七·经脉图考"` / `"甲一·亡者索引"`。

### 4.3 流派/境界/颜色的出现规则

- 涉及流派必用**双名**：写 "体修 / 爆脉流"，不要单写 "爆脉流"
- 涉及境界用**六阶原名**
- 涉及染色必是 11 种里的名（或明说"混元色"）
