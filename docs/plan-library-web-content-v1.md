## 0. 权威设定（云端**必须**在开工前精读并严格遵守）

### 0.1 境界体系：**六阶**，不是九阶

来源 `docs/worldview.md` L64-71，`plan-cultivation-v1.md` L132-139：

| 阶 | 名 | 代号 | 真元上限 |
|---|---|---|---|
| 0 | **醒灵** | Awaken | 10 |
| 1 | **引气** | Induce | 30 |
| 2 | **凝脉** | Condense | 80 |
| 3 | **固元** | Solidify | 150 |
| 4 | **通灵** | Spirit | 300 |
| 5 | **化虚** | Void | 500 |

化虚是天花板（worldview L80），全服只容 1-2 人。主流人群处于凝脉-固元段。**禁止**出现"筑基/金丹/元婴/化神/分神/飞升"等传统仙侠术语。

### 0.2 战斗流派：**四攻三防七流**，每个都是**双名**

来源 worldview L204-259。写作时**必须用双名**，不要简写：

- 攻击四流：**体修 / 爆脉流**、**器修 / 暗器流**、**地师 / 阵法流**、**毒蛊流**
- 防御三流：**截脉 / 震爆流**、**替尸 / 蜕壳流**、**绝灵 / 涡流流**

**核心约束**：末法时代**不存在外放护盾**，三防均针对"已入体"的攻击（L236-252）。写"护体真气罩"违反世界观。

### 0.3 真元染色：**11 种**（含混元色）

来源 worldview L294-329，plan-cultivation L207-217：

| 中文 | 代号 | 来源修习 |
|---|---|---|
| 锋锐色 | Sharp | 剑修 |
| 沉重色 | Heavy | 拳修/体修 |
| 温润色 | Mellow | 炼丹师 |
| 凝实色 | Solid | 炼器师 |
| 飘逸色 | Light | 御剑/驭物 |
| 缜密色 | Intricate | 阵法师 |
| 平和色 | Gentle | 医道 |
| 阴诡色 | Insidious | 毒蛊师 |
| 暴烈色 | Violent | 雷法 |
| 浊乱色 | Turbid | 吞噬/魔功 |
| 混元色 | 特殊 | 均匀全修（每单项 ≤25%） |

颜色与修习路径绑定，不是出生属性，约需 ~10h 实修才出"主色调"（L310）。

### 0.4 灵压三域 / 经脉拓扑

**三域**（L29-54）：
- 馈赠区（灵气 > 0）：修炼地
- 死域（灵气 = 0）：真元自然流失
- 负灵域（灵气 < 0）：天地反吸真元，境界越高吸越快；**战术价值**：通灵+可躲天劫、低阶可诱高阶进入实现反杀

**经脉**：12 正经（手足三阴三阳）+ 8 奇经（任督冲带 + 阴阳维跷）= 20 条。任督开通大幅抬升真元上限。

### 0.5 骨币经济：**半衰期指数衰减**

来源 worldview L515-537：
- 封灵骨币是唯一硬通货（修士注真元入异变兽骨制成）
- "一枚含 10 点真元的新币，箱子里放一个月可能只剩 2 点"（L520）—— 半衰期，不是线性扣减
- 真正保值：**信息**（灵眼坐标、闭关点、验证过的死亡遗念）。全世界"骨币会贬，记载不朽"（网站 tagline）
- 灵物操作磨损（L485-488）：箱子取出/背包转移扣 1%-5%，是天道的隐性交易税
- **禁止**出现"灵石"—— 唯一硬通货是骨币

### 0.6 死亡三机制

来源 worldview L640-790：
- **运数期**（前 3 次死亡）：100% 重生，但需满足 4 条件（24h 内未死 / 不在死域负灵域 / 业力低 / 有灵龛归属）；任一不满足直接进概率期
- **劫数期**（第 4 次起）：`P(重生) = max(5%, 80% - 15%×(n-3))`。第 4 死 65%、第 7 死 20%、第 8+ 保底 5%
- **遗念**（L748-771）：死亡瞬间天道生成，境界越高越详细；"天道不屑编造谎言"
- **终结与亡者博物馆**（L694-738）：角色死透后生平卷归档进亡者博物馆，公开查阅 —— **library-web 本身就是亡者博物馆**。这决定了馆藏叙事视角

### 0.7 六个地理区域（含北荒）

worldview L844-851：初醒原（0,0）/ 青云残峰（-3000,-2000）/ 灵泉湿地（-2500,2500）/ 血谷（3000,-2500）/ 幽暗地穴（2000,3000）/ 北荒（0,-7000）。**没有坐骑、没有传送**（L853）。

## 1. 风格锚（直接引用基准）

云端写作时以下三条作为**语气压舱**：

1. **tagline**（`library-web/src/layouts/Base.astro:17`）："骨币会贬，记载不朽"
2. **副标题**（`pages/index.astro:21-23`）："天地灵气将尽，唯文字不灭。此间所录，皆残土之真。"
3. **页脚**（`Base.astro:49`）："凡可立卷者，皆应先入馆藏，再论买卖与实现。"

**叙事立场**：library-web **即亡者博物馆本身**（worldview L736-738）。所有馆藏是"前人遗留，供后来者阅读"的第一手残篇 / 笔记 / 口信——**不是**第三人称百科。作者已死，语气陈述性冷漠，古意简省，带世界末日感但**不悲悯**。

**文体分布**（每本书指定一种，云端不要全写成长老论文）：
- 散修手札（最常见，带实地体温）
- 宗门长老注疏（学究、偏理论）
- 末法遗老追忆（缅怀灵气鼎盛，悲凉）
- 商贾农夫记账（接地气，柴米油盐）
- 死者遗书 / 临终口信（短、碎、带血气）
- 天道口吻摘录（冷漠陈述式，仅少量条目）

## 4. 写作硬约束

### 4.1 禁止词

- "灵石"（唯一硬通货是骨币）
- "九阶 / 筑基 / 金丹 / 元婴 / 化神 / 分神 / 飞升"（只有六阶）
- "护体真气罩 / 外放护盾"（末法无此物）
- "传送 / 瞬移 / 坐骑"（worldview L853 明确无）
- "系统 / 面板 / 属性点 / 升级"（现代游戏词汇）
- "AI / LLM / 由天道生成 / 我是..."（破戏）

### 4.2 格式硬要求

每册 md body **600-1200 字**（不计 frontmatter），分 2-4 节。

quote 字段 **20-40 字**（超过一行就是错的）。

`shelf` 字段建议格式 `"{天干/地支/数字} {方位/类别}·{子架}"`，如 `"丙三·币轨残片"` / `"乙七·经脉图考"` / `"甲一·亡者索引"`。

### 4.3 流派/境界/颜色的出现规则

- 涉及流派必用**双名**：写 "体修 / 爆脉流"，不要单写 "爆脉流"
- 涉及境界用**六阶原名**
- 涉及染色必是 11 种里的名（或明说"混元色"）

## 5. 验收标准

云端完成时：

1. **册数**：`src/content/books/*.md` ≥ **18 册**
2. **Frontmatter 完整**：所有 schema 字段非空，`hallSlug` ∈ 合法 5 值
3. **halls.json bookCount 更新**：匹配实际每馆册数
4. **构建通过**：`cd library-web && npm install && npm run build` 零错
5. **内容合规**：
   - 无 §4.1 禁止词
   - 境界名、流派名、染色名严格按 §0 权威设定
   - 不与 worldview.md 矛盾
6. **风格多样性**：18 册里至少 **4 种文体**（不要全写长老注疏）
7. **旧稿迁移保真**：8 本旧 JSON 的正文**不做文意改写**，只做字段映射与 markdown 排版
