# Bong · plan-playthrough-100h-gpt-v1 · 骨架

> 交叉验证标识：`-gpt`。本文件是 `docs/plans-skeleton/` 下的详细玩法路程骨架，**不是 Active plan**，不可被 `/consume-plan` 直接消费。

**主题**：从“普通人”到化虚的 100h 游玩路程。这里的“普通人”按正典解释为“无资源、无靠山、刚醒灵的普通散修”，不新增长期凡人境界；若需要真凡人教学，只作为 0-15 分钟序章。

**世界观锚点**：`worldview.md §三 修炼体系` · `§四 战斗系统` · `§五 七流派` · `§八 天道行为准则` · `§九 骨币/情报经济` · `§十 灵气零和` · `§十三 末法地理` · `§十六 活坍缩渊`

**代码实地锚点**：`server/src/cultivation/` · `server/src/lingtian/` · `server/src/alchemy/` · `server/src/forge/` · `server/src/world/tsy_*` · `server/src/world/extract_system.rs` · `server/src/world/tsy_container_search.rs` · `server/src/network/client_request_handler.rs` · `server/src/zhenfa_hooks.rs` · `client/src/main/java/com/bong/client/{alchemy,forge,lingtian,tsy}`

**交叉引用**：`plan-cultivation-v1` · `plan-skill-v1` · `plan-lingtian-v1` · `plan-alchemy-client-v1` · `plan-forge-leftovers-v1` · `plan-zhenfa-v1` · `plan-tribulation-v1` · `plan-tsy-v1` 与 TSY 子计划 · `plan-persistence-v1` · `plan-inventory-v1`

## §0 实地调研结论

| 模块 | 当前可依赖程度 | 结论 |
|---|---|---|
| 修炼 / 经脉 / 突破 | 高 | `Realm`、20 经脉、真元、污染、裂痕、淬炼、突破、渡虚劫入口已在 server；可作为 100h 主轴 |
| 灵田 / 采集 / 生态 | 高 | 开垦、种植、收获、补灵、偷灵、压力、道伥 spawn、客户端动作 UI 已有；可作为稳定资源轴 |
| 坍缩渊 / 撤离 | 中高 | 独立位面、负压、容器、搜刮 server event、死亡掉落、塌缩、撤离倒计时已有；client 撤离可用，容器搜刮 C2S/HUD 仍需补 |
| 炼丹 | 中 | server 内核、配方、炉、服丹、丹毒、放炉、skill XP 已有；投料/点火/取回与多炉客户端闭环仍在 `plan-alchemy-client-v1` |
| 炼器 | 中 | server 四步锻造、蓝图、结算、skill XP、放砧系统有基础；起炉/图谱翻页/客户端全流程仍在 `plan-forge-leftovers-v1` |
| 阵法 | 低 | `plan-zhenfa-v1` 是骨架，server 仅有 hook manifest 与灵田 `zhenfa_jvling` 预留；不可作为 100h 必经门槛 |
| 流派树 | 中低 | 爆脉/体修已有局部技能；暗器、毒蛊、截脉、蜕壳、涡流多为 skeleton；应做“路径倾向”而非完整职业锁 |
| Minecraft 开放玩法 | 中 | 方块、地形、灵田、炉/砧、世界探索、TSY 裂隙已接入；vanilla survival/crafting 不是主轴，需转译为修仙资源选择 |

**核心设计判断**：100h 主线必须以“经脉拓扑 + 灵气零和 + 世界探索风险”为硬轴，炼丹/炼器/TSY 是加速与变体，阵法与完整七流派先作为中后期扩展，不可写成当前必需闭环。

## §1 接入面

### §1.1 进料

- 从 `cultivation` 读取 `Realm`、`Cultivation.qi_current/qi_max`、`MeridianSystem`、`Contamination`、`QiColor`、`LifeRecord`。
- 从 `world/zone` 与 terrain raster 读取区域灵气、死域/负灵域、灵脉、伪灵脉、遗迹、TSY 裂隙。
- 从 `botany` / `lingtian` 读取野生植物、种子、田块、plot_qi、zone pressure、灵田 session。
- 从 `alchemy` 读取丹方、丹炉、服丹结果、丹毒、突破辅助 buff。
- 从 `forge` 读取蓝图、砧、锻造 session、成品质量、武器/工具产出。
- 从 `inventory` 读取背包格子、装备、骨币、材料、死亡掉落、物品磨损。
- 从 `world/tsy_*` 读取入场、负压、容器、搜刮、撤离、塌缩、秘境死亡掉落。
- 从 `skill` 读取 `herbalism/alchemy/forging` 技艺等级与经验。

### §1.2 出料

- 输出玩家每阶段目标：经脉数量、境界、必要资源、可选加速器、风险活动、恢复路径。
- 输出内容缺口清单，供后续 active plan 拆分：炼丹客户端、炼器客户端、TSY 搜刮客户端、阵法 runtime、持久化、境界名对齐。
- 输出玩法验收场景：0-2h、2-8h、8-22h、22-45h、45-70h、70-90h、90-100h 的 demo 路径。
- 输出平衡约束：材料最多缩短约 30%，丹药/法器不能绕过经脉拓扑，灵田不能打印无限灵气，TSY 死亡不能删除 20h 进度。

### §1.3 共享类型 / event

- 复用 `Realm`，但中文文案必须显示正典名：醒灵 / 引气 / 凝脉 / 固元 / 通灵 / 化虚。
- 复用 `MeridianId` 20 条经脉，不新增传统“炼气/筑基/金丹”等等级。
- 复用 `BreakthroughRequest`、`MeridianTarget`、`ForgeRequest`、`AlchemyOutcomeEvent`、`SkillXpGain`、`StartExtractRequest`、`StartSearchRequest`。
- 复用 `ColorKind` 作为染色长期沉淀，不把它和 `Contamination` 混成一个系统。
- 复用 `ZonePressureCrossed` 作为天道密度阈值入口，阵法后续也应往同一压力模型写权重。

### §1.4 跨仓库契约

- server：`server/src/cultivation/components.rs`、`breakthrough.rs`、`tick.rs`、`lingtian/systems.rs`、`alchemy/*`、`forge/*`、`world/tsy_*`、`network/client_request_handler.rs`。
- agent/schema：`agent/packages/schema/src/{client-request,server-data,cultivation,alchemy,forge,tsy,botany}.ts`。
- client：`InspectScreen`、`AlchemyScreen`、`ForgeScreen`、`LingtianActionScreen`、`ExtractInteractionBootstrap`、`BongHud`、`ClientRequestSender`。
- worldgen：terrain raster 的 `qi_density/mofa_decay/qi_vein_flow/TSY layers/ruin` 语义层。

### §1.5 代码正典差异待对齐

- 当前 `Realm` 注释仍是“觉醒/引灵/凝气/灵动/虚明”，应统一成“醒灵/引气/凝脉/通灵/化虚”。
- 当前代码 `required_meridians()` 为 `0/1/4/8/14/20`，正典表偏 `1/3/6/12 + 奇经`；本 skeleton 的可玩里程碑先按代码值做验收，另列对齐 TODO。
- `lingtian/mod.rs` 文件头注释仍写“不含补灵/收获/偷灵/密度阈值/客户端 UI”，但代码与 plan 已显示这些切片已落；需要后续整理注释，避免 agent 误读。

## §2 设计轴心

- [ ] **主线唯一硬门槛 = 经脉拓扑**：境界推进必须来自开脉、经脉完整度、心境、突破事件、渡虚劫。
- [ ] **100h 不是挂机打坐**：约 50h 是修炼内核，其余时间来自找地、采集、种田、炼丹炼器、搜打撤、受伤恢复、社交冲突与天道事件。
- [ ] **慢修与冒险并存**：每阶段都有安全慢修路线，也有 TSY/炼丹/炼器/抢灵田等高风险加速路线。
- [ ] **流派不是职业锁**：玩家差异来自经脉路径、真元染色、顿悟、装备与资源习惯；不做“选职业后锁技能树”。
- [ ] **丹器田阵是资源策略，不是升级按钮**：丹药/法器/灵田/阵法只能降低成本或改变打法，不能替代境界条件。
- [ ] **TSY 是高风险补课**：可以追赶资源差距，但死亡掉落和撤离读秒必须让玩家算账。
- [ ] **Minecraft 空间必须有意义**：建家、选址、洞穴、遗迹、灵脉、死域、负灵域、TSY 裂隙都影响路线，不做纯菜单修仙。
- [ ] **天道不发任务**：它只通过灵物密度、劫气、伪灵脉、道伥、域崩、渡虚劫改变环境。

| 时间 | 阶段 | 代码境界 | 代码门槛 | 玩家目标 | 核心体验 |
|---:|---|---|---:|---|---|

| 0.25-2h | 醒灵求活 | Awaken | 0→1 | 找第一处馈赠区，设第一条经脉目标 | 学会灵气零和、不能空真元、死域安全但没资源 |
| 2-8h | 引气立足 | Induce | 1→4 | 建第一个临时据点，开基础灵田 | 初次采药、补灵、偷灵、爆脉/污染教学 |

## §4 阶段细化

### §4.1 0-0.25h · 凡身序章

**定位**：把“普通人”处理成世界进入体验，不新增境界。

| 目标 | 内容 |
|---|---|
| 教学目标 | 看见灵气浓淡、真元条、背包格子、骨币半衰提示、死域/馈赠区差异 |
| Minecraft 操作 | 走路、采普通资源、看地形、认方块、拿起第一把锄/采药刀 |
| 末法规则 | 原地乱坐会吸干一小片地；真元不是自动回蓝；夜里远行有风险 |
| 退出条件 | 玩家第一次设置 `MeridianTarget` 或完成“找一块可坐之地” |

**不能做**：不要设计“凡人练到醒灵 10 小时”；不要发天道新手任务；不要给稳定新手礼包。

### §4.2 0.25-2h · 醒灵求活

**硬目标**：打开第一条经脉，进入引气前后。

| 玩法 | 路线 |
|---|---|
| 修炼 | 选择第一条正经目标，静坐吸正灵气，观察区域灵气下降 |
| 采集 | 采 `ci_she_hao` / 普通药草，学会手动采集比自动更稳 |
| 背包 | 认识 1×1 药草、骨币、工具耐久、丢弃与拾取 |
| 战斗 | 遭遇低阶野兽或玩家，体验污染/伤口/逃跑优先 |
| 开放世界 | 在 200-500 格内找水源、土、洞口、残灰边界，决定是否临时扎营 |

**慢修路线**：在低密度馈赠区静坐 + 少量采草，约 1.5-2h 达成。

**冒险加速**：偷吸别人空田 plot_qi、进入浅层危险地采高质草、打低阶野兽换材料，约 0.7-1.2h 但可能死亡/污染。

**验收场景**：新号 30 分钟内能完成“找馈赠区 → 设经脉目标 → 真元上涨 → 区域灵气下降 → 第一次突破请求”。

### §4.3 2-8h · 引气立足

**硬目标**：从 1 条推进到 4 条左右，形成第一个资源循环。

| 玩法 | 路线 |
|---|---|
| 修炼 | 经脉邻接选择开始重要，不能乱开导致后续路径低效 |
| 灵田 | 开垦 1-3 块田，种 `ci_she_hao_seed` / `ning_mai_cao_seed`，体验补灵冷却 |
| 炼丹 | 可服低阶回元/开脉辅助，但炼丹全链路未完成前先以 loot/交易获得丹药 |
| 炼器 | 使用基础工具，不把锻造作为硬门槛；可学习图谱残卷但不强制起炉 |
| TSY | 只允许浅层“看一眼”：进裂缝、真元被抽、找到出口、撤离 |
| MC 开放 | 建第一个隐蔽小屋，学习距离、坐标、死信箱、藏箱风险 |

**流派萌芽**：爆脉玩家偏 `flow_rate`；丹田玩家偏采集/灵田；器修玩家开始攒矿/骨；毒蛊与阵法只出现传闻和材料，不开放完整闭环。

**失败恢复**：死亡降境/空真元后，玩家应能靠安全采集 + 静坐在 30-60 分钟内回到原进度附近。

### §4.4 8-22h · 凝脉分路

**硬目标**：推进到 8 条左右，准备固元。

| 路线 | 核心循环 | 代码依赖 | 设计状态 |
|---|---|---|---|
| 体修 / 爆脉 | 淬炼流速、近身爆发、战后养伤 | `burst_meridian`、combat、meridian cracks | 最适合作为首个完整战斗流派 |
| 丹田经营 | 灵田 → 药材 → 丹药 → 开脉/恢复 | `lingtian` + `alchemy` | 灵田可玩，炼丹客户端需补齐 |
| 器修 | 采矿/兽骨 → 锻造工具/武器 → 探索效率 | `forge` + inventory | server 可算，起炉 UI 需补 |
| TSY 搬运工 | 低阶进浅层，带凡物，搜容器，读秒撤 | `tsy_container_search` + `extract_system` | server 有搜刮，client 请求需补 |
| 地师预备 | 看地、藏箱、残灰脚印、预留阵眼材料 | `zhenfa_hooks` | 不做硬门槛，只收集材料与知识 |

**阶段矛盾**：玩家第一次发现“最优修炼点会被吸干”，必须开始搬家、分仓、偷灵、交易或下 TSY。

**关键体验**：第一次在 TSY 里“多搜一具干尸还是立刻撤”；第一次炼丹炸炉或服丹丹毒；第一次锻造出低质但可用工具。

### §4.5 22-45h · 固元成家

**硬目标**：从 8 条推进到 14 条左右，建立长期资源线。

| 玩法 | 内容 |
|---|---|
| 修炼 | 经脉完整度、心境、污染管理开始比单纯开脉更重要 |
| 灵田 | 5-12 块田，出现 zone pressure，过度补灵触发道伥或清 plot_qi |
| 炼丹 | 需要完整投料/点火/取回 UI 后成为稳定加速器；否则以丹药 loot/交易替代 |
| 炼器 | 工具与武器开始影响探索半径；灵木苗与矿物进入长期规划 |
| 经济 | 骨币半衰迫使玩家把财富转成丹方、坐标、残卷、材料缓存 |
| TSY | 浅层稳定撤离，中层尝试；死亡掉落开始足够痛 |
| MC 开放 | 建主基地 + 1-2 个假仓/死信箱，不做传送，路线信息变资产 |

**固元门槛感**：不是“数值到了点按钮”，而是玩家必须证明自己能维护资源线、处理污染、承受一次失败。

**天道压力**：大型灵田、高级炉/砧、箱子集中、连续突破都会让负面概率上升。

### §4.6 45-70h · 通灵入局

**硬目标**：奇经推进到接近全通，玩家从“地区玩家”变成“全服变量”。

| 玩法 | 内容 |
|---|---|
| 修炼 | 奇经开启，真元池变大，但负灵域/TSY 绝对抽取也更凶 |
| 流派 | 染色主色明显：沉重/凝实/温润/缜密/阴诡等开始产生 inspect 可读差异 |
| 灵田 | 高阶作物周期长、压力高，必须分散种植或牺牲安全性 |
| 炼丹 | 固元丹/清浊/回元/突破辅助变成战略物资，丹方比成丹更值钱 |
| 炼器 | 成品不绑定，可抢可卖；耐久与维护成本进入平衡核心 |
| TSY | 深层不再对高境友好，通灵玩家反而倾向守浅层、抢低阶搬运工 |
| 社交 | 信息交易、截胡、报复、匿名 LifeRecord 成为主要叙事来源 |

**核心反转**：境界越高，不是越安全，而是越会被天道、玩家、负灵域、资源压力盯上。

### §4.7 70-90h · 半步化虚

**硬目标**：20 经脉全通，准备渡虚劫，但可以选择长期不渡。

| 准备项 | 内容 |
|---|---|
| 地点 | 找 2-3 个候选渡劫点：有路可跑、灵气足、远离仇家基地、可布置但不安全 |
| 资源 | 回元丹、清浊类、抗污染手段、备用武器、低灵质逃生包 |
| 信息 | 假坐标、死信箱、盟友/诱饵、敌人在线规律、TSY 裂隙路线 |
| 阵法 | 若 `plan-zhenfa-v1` 落地，可作为欺天/防御/诡雷辅助；未落地时只保留叙事需求 |
| 分仓 | 防止被截胡一次清空 50h 资产，遵守灵物密度阈值 |

**玩家选择**：通灵圆满可以不渡，继续当地区强者；化虚是服务器级风险，不是常规毕业按钮。

### §4.8 90-100h+ · 渡虚窗口

**硬目标**：通灵 → 化虚只通过渡虚劫。

| 流程 | 内容 |
|---|---|
| 起劫 | 玩家主动确认，全服预兆广播，坐标按 POI 精度公开 |
| 锁定 | 不能取消，不能安全下线，截胡者有 10-20 分钟地理赶路窗口 |
| 劫波 | 雷劫 + 心魔 + PVP 截胡叠加，天道不偏袒任何人 |
| 成功 | 若化虚名额未满则进入化虚；名额满则半步化虚 |
| 失败 | 天劫失败退回通灵初期；被玩家截胡则走正常死亡与掉落 |

**化虚后**：不是“毕业安全”，而是高消耗、高注视、高维护状态；服务器内仅少数人能维持。

## §5 不同流派的 100h 体验线

### §5.1 爆脉体修线

| 阶段 | 体验 |
|---|---|
| 0-8h | 省材料，靠近战和逃跑；第一次爆脉造成经脉裂痕 |
| 8-22h | `beng_quan` / 贴脸输出成型，偏淬炼 `flow_rate` |
| 22-45h | 战后修养成为成本，清浊/回元丹变刚需 |
| 45-70h | 强截胡能力，但被暗器/毒蛊/涡流反制 |
| 70-100h | 渡劫时抗雷强，但容易被多人放风筝消耗 |

**代码优先级**：体表伤口 ↔ 经脉裂痕桥接、爆脉长期代价、技能 UI 反馈。

### §5.2 丹修 / 灵田线

| 阶段 | 体验 |
|---|---|
| 0-8h | 开小田，学会 plot_qi 和补灵冷却 |
| 8-22h | 通过药材降低开脉风险，温润色开始积累 |
| 22-45h | 田块分散与压力管理成为核心，丹方情报变资产 |
| 45-70h | 高阶丹药支撑团队或交易，丹毒管理限制滥用 |
| 70-100h | 渡劫准备者最需要丹修，但丹修基地也最容易被盯 |

**代码优先级**：`plan-alchemy-client-v1` 的多炉路由、投料/点火/取回、炸炉反噬、正式配方正典化。

### §5.3 器修 / 炼器线

| 阶段 | 体验 |
|---|---|
| 0-8h | 攒矿、兽骨、工具耐久；战斗不强但探索效率好 |
| 8-22h | 初级锻造出工具/武器，凝实色倾向出现 |
| 22-45h | 装备可交易可掉落，维护和材料压力成核心代价 |
| 45-70h | 暗器/重狙分支需要载体、封元、射空损失 |
| 70-100h | 渡劫装备不保命，只提供少数关键窗口 |

**代码优先级**：`ForgeStartSession` C2S、图谱翻页、forge item runtime 字段、锻造 UI 三阶段、持久化。

### §5.4 地师 / 阵法线

| 阶段 | 体验 |
|---|---|
| 0-22h | 只做“看地”和材料储备，学习残灰脚印、通道、阵眼位置 |
| 22-45h | 若 P0 落地，单点真元诡雷可作为防田/防仓工具 |
| 45-70h | 聚灵阵/防御阵必须接天道压力，不可变无限灵气机 |
| 70-100h | 欺天阵可作为渡虚劫前策略，但失败反噬更重 |

**代码红线**：当前不可作为主线必需；不做传送阵；阵法必须朽坏、可破、可反噬。

### §5.5 毒蛊 / 暗杀线

| 阶段 | 体验 |
|---|---|
| 0-22h | 主要是材料和传闻，不能早期开放永久折损真元上限 |
| 22-45h | 单针/脏真元作为高风险 PVP 工具，暴露代价极高 |
| 45-70h | 克高境，适合追踪战和资源战，但自身经脉慢性侵蚀 |
| 70-100h | 可截胡渡劫者，但被抓到基本全服追杀 |

**代码优先级**：special contamination 维度、解蛊、身份暴露、社交后果。

### §5.6 混元开放探索线

| 阶段 | 体验 |
|---|---|
| 0-22h | 不极端专精，靠地图知识、路线和小额交易活下去 |
| 22-45h | 做情报中介、TSY 轻装搬运、灵田偷灵、低风险修炼 |
| 45-70h | 不一定战斗强，但掌握裂隙、灵眼、伪灵脉时刻表 |
| 70-100h | 可以成为渡劫事件的情报商、诱饵、截胡协作者 |

**设计意义**：Minecraft 开放玩法的代表，不把所有玩家都赶进战斗职业。

## §6 炼器 / 炼丹 / 种田 / 阵法 / TSY 的分工

| 系统 | 100h 中的定位 | 不能做 | 可做 |
|---|---|---|---|
| 炼丹 | 降低突破失败、恢复真元、处理污染、交易高价值信息 | 直接跳境界、无毒无限吃 | 30% 内加速、丹毒、炸炉、残卷情报 |
| 炼器 | 提升探索/战斗效率，承载器修与工具维护 | 装备绑定、神器常驻、远程法术对波 | 可掉落可抢、低耐久遗物、载体封元 |
| 灵田 | 中期稳定器，给丹药与交易供料 | 大农庄安全打印、无限灵气 | 分散田、偷灵、压力、道伥、补灵冷却 |
| 阵法 | 中后期空间策略与天道博弈 | 传送阵、永久机器、主线硬门槛 | 真元诡雷、聚灵但有压力、欺天但可反噬 |
| TSY | 高风险追赶与上古遗物来源 | 奖励房、定时副本、原地传送撤离 | 负压秒表、容器搜刮、读秒撤离、死亡掉落 |
| MC 开放 | 所有系统的空间载体 | 纯菜单修仙、无距离成本 | 地形选址、洞穴遗迹、建筑藏匿、路线信息 |

## §7 Minecraft 开放玩法落点

- [ ] **建家不是装饰**：基地位置由灵气、死域安全性、水源、洞穴、TSY 裂隙、灵田压力共同决定。
- [ ] **采矿不是平行玩法**：矿物进入炼器、阵眼、工具耐久、交易，不单独走 vanilla 科技树。
- [ ] **种田不是农场模拟**：每块田从区域或材料扣灵气，聚集过多会被天道统计。
- [ ] **探索不是跑图开点**：远行 10-20 分钟成本真实存在，坐标、路线、死亡遗念、残灰脚印是核心资产。
- [ ] **建筑会暴露**：高价值炉、砧、箱子、灵田、阵法都增加被玩家或天道发现的概率。
- [ ] **没有传送缓解成本**：TSY 入口/出口是风险点，不是交通系统；阵法也不做传送阵。
- [ ] **死亡是学费**：玩家丢资源、寿元/境界受损，但获得遗念、敌人情报、路线知识。

## §8 数值与平衡初稿

### §8.1 境界时间分配

| 境界段 | 建议总时长 | 纯修炼占比 | 横向玩法占比 | 失败恢复预算 |
|---|---:|---:|---:|---:|
| 醒灵 | 2h | 50% | 40% | 10% |
| 引气 | 6h | 45% | 40% | 15% |
| 凝脉 | 14h | 40% | 45% | 15% |
| 固元 | 23h | 35% | 45% | 20% |
| 通灵前中 | 25h | 35% | 40% | 25% |
| 半步化虚 | 20h | 25% | 45% | 30% |
| 渡虚窗口 | 10h+ | 20% | 50% | 30% |

### §8.2 加速上限

- 丹药、材料、法器、灵田、TSY loot 对突破成功率或开脉速度的总收益建议 clamp 在约 30%。
- 高风险活动可以节省时间，但不能绕过：经脉数量、完整度、心境、污染、渡虚劫。
- 单次死亡不应删除 20h 进度；秘境所得可全掉，角色拓扑进度应有恢复路径。
- 骨币半衰让玩家少囤货，多交易信息与消耗品。

### §8.3 风险梯度

| 风险 | 适合阶段 | 代价 | 奖励 |
|---|---|---|---|
| 静坐吸灵 | 全期 | 地区灵气下降、被发现 | 稳定开脉 |
| 野外采集 | 0-45h | 暴露路线、受伤、材料腐败 | 药材/种子/信息 |
| 灵田经营 | 2-70h | 被偷、压力、道伥 | 稳定材料 |
| 炼丹炼器 | 8-100h | 炸炉、废料、材料损耗 | 加速、装备、交易资产 |
| TSY 浅层 | 8-45h | 真元抽吸、死亡掉落 | 追赶资源 |
| TSY 深层 | 45-100h | 高境更危险、撤离更久 | 遗物/钥匙/高价值残卷 |
| 渡虚劫 | 90h+ | 全服广播、截胡、退境 | 化虚名额 |

## §10 验收场景

### §10.1 新号 2h 验收

- [ ] 进入世界后 15 分钟内能理解灵气/真元/死域/馈赠区。
- [ ] 能设定第一条经脉目标并观察开脉进度。
- [ ] 能采集一种药材，放入背包，并看到物品/质量/堆叠。
- [ ] 能开垦一块田，播种，看到 session HUD。
- [ ] 能经历一次小风险：污染、受伤、被抢、真元见底或逃跑。

### §10.2 22h 凝脉验收

- [ ] 至少三条路线可到凝脉：慢修种田、TSY 冒险、战斗掠夺。
- [ ] 玩家有至少一个明显倾向：爆脉/丹田/器修/搬运/情报。
- [ ] TSY 完成一次完整“进 → 搜 → 撤 → loot 入包”。
- [ ] 炼丹或炼器至少一个完成全链路，另一个可作为交易/loot 替代。

### §10.3 45h 固元验收

- [ ] 玩家拥有一个可被攻击、可被偷、可搬迁的长期资源点。
- [ ] 大型灵田或高灵物仓储触发过至少一次天道压力反馈。
- [ ] 死亡/失败后可恢复，但明显损失时间、资源或寿元。
- [ ] 骨币半衰迫使玩家进行消费、交易或信息转化。

### §10.4 70h 通灵验收

- [ ] 高境玩家进入 TSY 深层比低境搬运工更痛，形成反直觉策略。
- [ ] Inspect 能读到玩家染色/流派倾向，但不是职业标签。
- [ ] 至少两种流派之间有真实反制：爆脉贴脸、器修远射、涡流反载体、毒蛊耗高境。
- [ ] 玩家能通过地理与情报影响战斗结果，而不是只比数值。

### §10.5 100h 化虚验收

- [ ] 通灵圆满玩家可以选择不起劫，继续作为地区强者生存。
- [ ] 起劫会触发全服级可感知事件，其他玩家可赶路、观战、截胡或避难。
- [ ] 天劫失败和玩家截胡失败走不同惩罚路径。
- [ ] 成功化虚后仍受灵气零和、天道注视、社会敌意制约。

## §11 世界观红线

- [ ] 不新增传统境界：炼气、筑基、金丹、元婴等只能作为骗局/旧称。
- [ ] 不新增无成本灵气来源；聚灵、补灵、种植、炼丹都必须记账。
- [ ] 不把天道写成任务 NPC、导师、善恶裁判或奖励发放者。
- [ ] 不把骨币写成稳定货币；价值按剩余真元，不按枚数。
- [ ] 不把灵石写成货币；灵石是燃料。
- [ ] 不做传送阵，不用传送抹平 2000-5000 格世界距离。
- [ ] 不做装备绑定；器物可交易、可掉落、可抢。
- [ ] 不让丹药、法器、阵法绕过经脉拓扑和渡虚劫。
- [ ] 不把污染和真元染色混成同一数值。
- [ ] 不把未实现系统写成已实装，尤其是阵法 runtime 与炼丹/炼器客户端全链路。
